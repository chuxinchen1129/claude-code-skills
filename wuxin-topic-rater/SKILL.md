# 悟昕选题评分器

**Skill名称**：wuxin-topic-rater

**用途**：对选题进行多维度评分，筛选高质量选题

**适用场景**：
- 话题池筛选
- 选题质量评估
- A/B测试选题选择

---

## 快速开始

用户触发本Skill时，请求以下必填信息：

```
请提供以下信息：
1. 候选选题文件路径（JSON/Excel）
2. 营销节点（当前营销节日）
3. 目标数量（希望筛选出的选题数量，如16个）
```

可选信息：
4. 及格分数线（默认20分）
5. 淘汰规则（自定义淘汰条件）

---

## 工作流程

### 步骤1：读取候选选题

支持输入格式：
- JSON文件（来自wuxin-topic-pool-generator）
- Excel文件（手动整理的选题）

### 步骤1.5：场景分布比例讨论 ⭐ NEW

**目的**：分析候选选题的场景分布，与用户讨论场景比例是否合理

**执行内容**：

1. **分析当前场景分布**
   - 统计各场景类型的数量和占比
   - 识别场景过于集中或缺失的场景类型
   - 显示场景分布数据

2. **生成场景分布建议**
   根据营销节点类型（送礼/科普/促销）生成场景建议
   - 提供推荐场景比例
   - 说明理由

3. **与用户讨论**
   - 使用 AskUserQuestion 工具征求用户意见
   - 让用户选择：接受建议、自定义比例、跳过调整

**示例**：

```markdown
## 场景分布分析

### 当前候选选题的场景分布

| 场景类型 | 数量 | 占比 |
|---------|------|------|
| 送礼场景 | 20 | 40% |
| 更年期场景 | 5 | 10% |
| 早醒场景 | 5 | 10% |
| 入睡困难场景 | 5 | 10% |
| 子女关心场景 | 5 | 10% |
| 对比场景 | 5 | 10% |
| 情感共鸣场景 | 5 | 10% |

**观察**：送礼场景占比40%，场景分布不够均衡

---

### 建议场景分布（母亲节送礼节点）

| 场景类型 | 建议占比 | 理由 |
|---------|----------|------|
| 送礼场景 | 40-50% | 母亲节以送礼为主 |
| 使用感受场景 | 20-30% | 增加产品效果体现 |
| 情感共鸣场景 | 15-25% | 加强情感连接 |
| 其他场景 | 10-20% | 丰富内容多样性 |

---

**请选择操作**：

1. 接受建议并按比例筛选
2. 自定义场景比例
3. 跳过调整，直接评分
```

**配置使用方式**：

1. **自动匹配**：根据营销节点关键词自动选择配置
   - 包含"母亲节/送礼" → 母亲节配置
   - 包含"女神节/美妆" → 女神节配置
   - 包含"520/表白/浪漫" → 520配置
   - 包含"父亲节/爸爸" → 父亲节配置
   - 包含"睡眠日/世界睡眠/科普" → 世界睡眠日配置

2. **动态调整**：用户可以接受建议后微调比例

3. **自定义模式**：用户可以完全自定义场景分布

---

**用户确认后**：
- 如果选择"接受建议"，按建议比例重新筛选选题
- 如果选择"自定义比例"，询问用户具体的比例要求
- 如果选择"跳过调整"，继续执行步骤2的6维度评分

### 步骤2：6维度评分（每个维度1-5分）

**维度1：营销节点关联度 ⭐⭐⭐**
**权重**：⭐⭐⭐（最高权重）

| 得分 | 说明 | 检查项 |
|------|------|--------|
| 5分 | 强关联，多次出现营销节点关键词，自然融入 | ✅ 标题包含、✅ 描述包含 |
| 4分 | 明显关联，出现营销节点关键词 | ✅ 标题包含 |
| 3分 | 有关联，营销节点关键词出现但不突出 | ⚠️ 标题包含 |
| 2分 | 弱关联，营销节点关键词极少 | ⚠️ 标题包含 |
| 1分 | 无关，完全未提及营销节点 | ❌ 标题不含 |
| 0分 | 淘汰项，与营销节点完全无关 | ❌ 直接淘汰 |

**维度2：场景多样化 ⭐⭐**
**权重**：⭐⭐

| 得分 | 说明 | 检查项 |
|------|------|--------|
| 5分 | 场景独特，与其他选题完全不同 | ✅ 场景分类唯一 |
| 4分 | 场景有一定差异 | ✅ 角度不同 |
| 3分 | 场景常见但角度不同 | ⚠️ 相同场景但切入不同 |
| 2分 | 场景重复度高 | ❌ 场景完全相同 |
| 1分 | 场景严重重复 | ❌ 多个选题相同场景 |
| 0分 | 淘汰项，场景与其他选题完全相同 | ❌ 直接淘汰 |

**维度3：钩子强度 ⭐⭐**
**权重**：⭐⭐

| 得分 | 说明 | 检查项 |
|------|------|--------|
| 5分 | 钩子极强，数据冲击+痛点直击 | ✅ 15-25字、有冲击力 |
| 4分 | 钩子强，有明显冲击力 | ✅ 有吸引力 |
| 3分 | 钩子一般，有一定吸引力 | ⚠️ 勉强关注 |
| 2分 | 钩子弱，缺乏冲击力 | ❌ 缺乏钩子 |
| 1分 | 钩子极弱，无法抓住注意力 | ❌ 无法抓住注意力 |
| 0分 | 淘汰项，无钩子 | ❌ 直接淘汰 |

**维度4：情感共鸣 ⭐**
**权重**：⭐

| 得分 | 说明 | 检查项 |
|------|------|--------|
| 5分 | 强共鸣，痛点+情绪表达精准 | ✅ 痛点具体、✅ 情绪词汇 |
| 4分 | 明显共鸣，痛点描述具体 | ✅ 痛点具体 |
| 3分 | 一般共鸣，痛点描述可以 | ⚠️ 痛点可理解 |
| 2分 | 弱共鸣，痛点模糊 | ❌ 痛点抽象 |
| 1分 | 极弱共鸣，无痛点代入感 | ❌ 无代入感 |
| 0分 | 淘汰项，无情感连接 | ❌ 直接淘汰 |

**维度5：品牌一致性 ⭐**
**权重**：⭐

| 得分 | 说明 | 检查项 |
|------|------|--------|
| 5分 | 完整包含7大卖点+3大支柱 | ✅ 全部包含 |
| 4分 | 包含大部分卖点，核心卖点齐全 | ✅ 核心卖点都有 |
| 3分 | 包含部分卖点，核心卖点缺失 | ⚠️ 部分包含 |
| 2分 | 卖点少，品牌信息模糊 | ❌ 卖点缺失 |
| 1分 | 卖点极少，品牌信息严重缺失 | ❌ 严重缺失 |
| 0分 | 淘汰项，无品牌卖点 | ❌ 直接淘汰 |

**维度6：数据支撑 ⭐**
**权重**：⭐

| 得分 | 说明 | 检查项 |
|------|------|--------|
| 5分 | 数据丰富，包含3个以上数据点 | ✅ 80%+、30%、75%、32秒、21天、160g |
| 4分 | 数据充足，包含2个数据点 | ✅ 80%+、30% |
| 3分 | 数据一般，包含1个数据点 | ⚠️ 有数据 |
| 2分 | 数据少，数据模糊 | ❌ 数据模糊 |
| 1分 | 数据极少，无具体数据 | ❌ 无数据 |
| 0分 | 淘汰项，无数据支撑 | ❌ 直接淘汰 |

### 步骤3：计算总分和评级

**总分计算**：
- 总分 = 各维度得分相加（满分30分）

**评级标准**：
- **淘汰**：营销节点关联度<3分 或 总分<20分
- **及格**：20-24分
- **优秀**：25-30分
