# 悟昕视频脚本生成器

**技能名称**：wuxin-script-generator

**用途**：生成30秒短视频脚本，支持话题池裂变、评分筛选、标准格式输出

---

## 功能概览

| 功能 | 输入 | 输出 |
|------|------|------|
| **话题池生成** | 营销节点+选题数量 | 候选话题列表(JSON/Excel) |
| **话题评分** | 候选话题+评分标准 | 高质量话题(Top N) |
| **脚本生成** | 高质量话题+品牌植入度 | 视频脚本(Excel/Markdown) |
| **完整流程** | 营销节点+目标数量 | 最终脚本Excel |

---

## 使用场景

### 场景1：完整流程（推荐）⭐

**用户说**：
- "生成常规投放脚本"
- "生成Top18视频脚本"
- "悟昕日常推广内容"

**自动执行**：
```
1. 话题池生成（10基础 × 5裂变 = 50选题）
   ↓
2. 话题评分筛选（6维度评分 → Top18）
   ↓
3. 脚本生成（标准格式12列）
   ↓
4. Excel输出
```

**输出**：`常规投放_Top18脚本_标准格式版.xlsx`

### 场景2：单步执行

**用户说**：
- "生成话题池：常规投放，10个基础选题"
- "评分筛选：常规投放_话题池.xlsx"
- "生成脚本：使用场景+午休场景"

---

## CLI 命令

### 完整流程
```bash
python src/main.py full-pipeline \
  --node "常规投放" \
  --target-count 18
```

### 单步命令
```bash
# 话题池生成
python src/main.py topic-pool --node "常规投放" --base-topics 10 --fission 5

# 话题评分
python src/main.py rate --input "常规投放_话题池.xlsx" --target-count 18 --passing-line 25

# 脚本生成
python src/main.py scripts --input "常规投放_Top18选题.xlsx" --brand-integration soft
```

## 品牌植入规范

### 软性植入原则

| 原则 | 说明 |
|------|------|
| **台词不提名** | 全程不说"悟昕"、"睡眠仪" |
| **产品后置** | 前20秒不出现产品画面 |
| **聚焦价值** | 讲睡眠问题，不讲产品功能 |
| **CTA软引导** | "搜好睡眠"而非"搜悟昕" |

### 品牌露出规范

| 时间 | 露出要求 |
|------|----------|
| **0-20s** | 禁止产品露脸 |
| **20-25s** | 可作为场景元素（背景） |
| **25-30s** | 可快速闪现（≤1秒） |
