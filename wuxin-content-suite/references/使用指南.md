## 快速开始

### 安装检查

确保以下路径存在：
- `03_WUXIN_CONTENT/assets/` - 共享资源库
- `skills/wuxin-script-generator/` - 视频脚本生成器
- `skills/wuxin-xhs-content/` - 科普图文生成器
- `skills/wuxin-pr-article/` - 公关文章生成器
- `skills/wuxin-wechat-article/` - 公众号文章生成器

### 方式一：使用套件统一入口（推荐）

```bash
cd skills/wuxin-content-suite

# 查看帮助
python cli.py --help

# 查看套件信息
python cli.py info

# 生成视频脚本
python cli.py generate --type script --node "常规投放" --count 18

# 生成科普图文
python cli.py generate --type xhs --node "常规投放" --days 7

# 生成公关文章
python cli.py generate --type pr --article-type brand-story

# 生成公众号文章
python cli.py generate --type wechat --article-type science-article --topic "深睡"
```

### 方式二：直接调用子技能

每个生成器都可以独立使用：

```bash
# 视频脚本生成器
cd skills/wuxin-script-generator
python src/main.py full-pipeline --node "常规投放" --target-count 18

# 科普图文生成器
cd skills/wuxin-xhs-content
python src/main.py batch --node "常规投放" --days 7

# 公关文章生成器
cd skills/wuxin-pr-article
python src/main.py brand-story

# 公众号文章生成器
cd skills/wuxin-wechat-article
python src/main.py science-article --topic "深睡"
```

## 内容类型详解

### 1. 视频脚本生成器

**用途**：生成30秒短视频脚本（4段式结构）

**输出格式**：Excel（12列标准格式）

**使用场景**：
- 抖音/视频号短视频投放
- 常规内容日更
- 营销节点专题

**核心特性**：
- 话题池生成（10个基础 × 5个裂变 = 50个候选）
- 6维度评分筛选（营销节点关联度、场景多样化、钩子强度等）
- 超软品牌植入（全程不提名产品，聚焦价值）
- CTA软引导（搜"好睡眠"而非"搜悟昕"）

**命令示例**：
```bash
# 完整流程（推荐）
python src/main.py full-pipeline --node "常规投放" --target-count 18

# 分步执行
python src/main.py topic-pool --node "常规投放"
python src/main.py rate --input "常规投放_话题池.xlsx" --target-count 18
python src/main.py scripts --input "常规投放_Top18选题.xlsx"
```

---

### 2. 科普图文生成器

**用途**：生成小红书科普图文内容

**输出格式**：CSV（13字段标准格式）

**使用场景**：
- 小红书笔记发布
- 图文内容投放
- 品牌科普宣传

**核心特性**：
- 营销日历驱动（自动匹配营销节点）
- 科学概念匹配（从Wiki库匹配）
- 标准内容结构（intro + 3板块 + CTA）
- 小红书风格（温暖治愈、科学循证）

**命令示例**：
```bash
# 批量生成（日期范围）
python src/main.py batch --node "常规投放" --days 7

# 单篇生成
python src/main.py single --topic "深睡" --audience "职场人士" --pain-point "睡够8小时还是累" --angle "深睡质量提升"
```

---

### 3. 公关文章生成器

**用途**：生成正式公关文章和软文

**输出格式**：Markdown

**使用场景**：
- 品牌背书文章
- 行业洞察文章
- 用户成功案例
- 媒体新闻稿件

**核心特性**：
- 正式公文风格
- 品牌权威背书
- 客观第三方视角
- 数据支撑观点

**命令示例**：
```bash
# 品牌背书文
python src/main.py brand-story

# 行业洞察文
python src/main.py industry-insight --topic "睡眠科技趋势"

# 用户故事文
python src/main.py user-story --audience "职场人士"

# 媒体稿件
python src/main.py media-release --event "新品发布" --angle "推出新款睡眠监测设备"
```

---

### 4. 公众号文章生成器

**用途**：生成公众号深度文章

**输出格式**：Markdown

**使用场景**：
- 公众号推文
- 深度科普文章
- 故事化内容
- 清单型文章

**核心特性**：
- 长文深度（2000字左右）
- 讲故事逻辑
- 排版友好
- 互动引导

**命令示例**：
```bash
# 科普长文
python src/main.py science-article --topic "深睡" --depth 2000

# 故事型文章
python src/main.py story-article --scene "失眠" --persona "职场妈妈"

# 清单型文章
python src/main.py list-article --topic "睡眠误区" --count 10
```

## 常见问题

### Q: 如何选择合适的生成器？

**A**: 根据发布平台和内容形式选择：
- 抖音/视频号 → 视频脚本生成器
- 小红书 → 科普图文生成器
- 媒体报道 → 公关文章生成器
- 公众号 → 公众号文章生成器

### Q: 套件入口和子技能有什么区别？

**A**:
- **套件入口**：统一调用接口，适合快速使用
- **子技能**：独立功能，支持更多参数和自定义

### Q: 输出文件在哪里？

**A**: 默认输出在各技能的 `output/` 目录下，可通过 `--output` 参数自定义。

### Q: 如何修改品牌规范？

**A**: 修改中心配置文件 `03_WUXIN_CONTENT/assets/config.yaml`，所有生成器会自动生效。
