## 架构概览

### 设计原则

1. **资源集中共享**：品牌资产、Wiki、模板统一管理
2. **技能松耦合**：每个生成器独立，可单独使用和升级
3. **配置外部化**：通过config.yaml统一管理参数
4. **接口标准化**：标准化的输入输出接口

## 数据接口标准

### 输入接口（标准化JSON）

```json
{
  "task_type": "script|xhs_content|pr_article|wechat_article",
  "marketing_node": "常规投放|春节送礼|母亲节",
  "target_audience": "职场人士|父母长辈|...",
  "content_count": 18,
  "brand_integration": "soft|medium|hard",
  "topics": [
    {
      "id": 1,
      "scene": "使用场景",
      "pain_point": "午休越睡越困",
      "angle": "高质量午休回血"
    }
  ]
}
```

### 输出接口（标准化）

```json
{
  "success": true,
  "task_type": "script",
  "content_count": 18,
  "output_files": [
    "常规投放_Top18脚本_标准格式版.xlsx",
    "常规投放_Top18脚本_标准格式版.md"
  ],
  "metadata": {
    "generated_at": "2026-02-26T10:00:00",
    "generator_version": "1.0.0"
  }
}
```

## 配置管理

### 中心配置 (03_WUXIN_CONTENT/assets/config.yaml)

```yaml
version: 1.0.0
updated: 2026-02-26

# 资源路径
resources:
  brand:
    selling_points: brand/selling-points.md
    knowledge_base: brand/knowledge-base.md
  wiki:
    topics: wiki/topics/
    guides: wiki/guides/
  templates:
    video_script: templates/video-script/
    xhs_content: templates/xhs-content/
    pr_article: templates/pr-article/
    wechat_article: templates/wechat-article/

# 默认参数
defaults:
  script:
    duration: 30
    structure: 4_segment
    brand_integration: soft
  xhs_content:
    sections: 3
    target_words: 500

# 品牌规范
brand_guidelines:
  tone_words: [温暖治愈, 科学循证, 优雅克制]
  forbidden_words: [家人们, 绝绝子, 根治, 第一]
  selling_points: [AI智能睡眠管理师, 医疗级EEG脑电监测, ...]
```

### 技能配置引用

每个技能的 config.yaml 通过 `include` 引用共享配置：

```yaml
# 引用共享配置
include: ../../03_WUXIN_CONTENT/assets/config.yaml

# 技能特定配置
defaults:
  # 覆盖或扩展默认值
  ...
```

## 品牌规范统一

### 语气要求

所有生成器遵循统一语气：
- **温暖治愈**：亲切友好的表达
- **科学循证**：基于数据和事实
- **优雅克制**：不夸张不造作

### 禁用词汇

统一避免以下词汇：
- 家人们、绝绝子（过于网红化）
- 根治、必需（绝对化表述）
- 第一（缺乏数据支撑）

### 核心卖点

统一使用以下核心卖点：
1. AI智能睡眠管理师
2. 医疗级EEG脑电监测
3. 监测-干预-反馈闭环
4. FDA认证CES技术
5. 80%医疗级准确率
6. 缩短30%入睡时间
7. 睡眠质量改善75%以上

## 触发词设计

### 统一触发词规范

| 内容类型 | 触发词 | 技能 |
|---------|--------|------|
| 视频脚本 | "生成视频脚本"、"悟昕脚本"、"30秒脚本" | wuxin-script-generator |
| 科普图文 | "生成科普图文"、"小红书图文"、"悟昕科普" | wuxin-xhs-content |
| 公关文章 | "生成公关文章"、"品牌软文"、"媒体稿件" | wuxin-pr-article |
| 公众号 | "生成公众号文章"、"微信推文"、"长文" | wuxin-wechat-article |

### 组合触发

在套件 SKILL.md 中定义组合触发词，支持自然语言识别内容类型。

## 扩展指南

### 新增内容类型

1. 在 `skills/` 下创建新技能目录
2. 创建 SKILL.md（定义触发词）
3. 创建 config.yaml（引用共享配置）
4. 创建 src/generator.py（实现生成逻辑）
5. 在套件 cli.py 中注册新类型

### 新增共享资源

1. 将资源放入 `03_WUXIN_CONTENT/assets/` 对应目录
2. 更新中心配置 `config.yaml`
3. 所有技能自动可用新资源

### 修改品牌规范

1. 修改 `03_WUXIN_CONTENT/assets/config.yaml`
2. 所有技能自动应用新规范
3. 无需逐个修改技能
