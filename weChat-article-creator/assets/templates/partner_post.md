# 合作方文章模板

## 适用场景
- 在合作方自媒体平台发布品牌内容
- 需要融合品牌信息与合作方风格
- 严格遵循品牌信息，适配合作方调性

## 核心挑战
**平衡**：品牌信息完整性 × 合作方风格适配度

## 核心原则
1. **品牌信息优先**：核心卖点必须完整、准确
2. **风格深度融合**：自然融入合作方表达风格
3. **尊重读者体验**：不破坏合作方读者期待
4. **双方价值平衡**：品牌推广 + 合作方内容价值

## 工作流程

### 阶段1：获取品牌档案
- 读取brand_profiles/[品牌名].md
- 确认本次推广核心信息
- 确认不可更改的核心卖点

### 阶段2：合作方信息收集
用户提供以下之一：
- 合作方自媒体账号/链接
- 3-5篇该自媒体过往文章
- 合作方风格说明

### 阶段3：合作方风格分析
**分析维度**：

**a. 开头方式**
- 对话式："大家好，我是XX"
- 引用式："正如XX所说......"
- 故事式："记得有一次......"
- 数据式："据最新研究显示......"
- 提问式："你是否也有过......"

**b. 语言风格**
- 严肃/活泼/专业/幽默
- 正式/口语化
- 情感化/理性化
- 句式长短偏好

**c. 段落结构**
- 长段落/短段落
- 段落数量
- 分段习惯

**d. 标题习惯**
- 长度（短标题/长标题）
- 用词（提问/陈述/感叹）
- 标题结构（单标题/主副标题）
- 是否用emoji

**e. emoji使用**
- 使用频率（密集/稀少/不用）
- 类型偏好（特定风格emoji）
- 位置习惯（标题/正文/结尾）

**f. 特殊标签或格式**
- 固定开头语
- 固定结尾语
- 特殊排版标记
- 话题标签习惯

**g. 内容价值倾向**
- 知识输出型
- 观点输出型
- 情感共鸣型
- 实用技巧型

**输出**：风格分析报告（可选择保存到partner_analysis/[合作方名]/）

---

### 阶段4：确定切入点
**考虑因素**：
- 合作方受众特点
- 合作方内容调性
- 品牌信息适配度

**切入方式选择**：

**a. 知识科普型**
- 适合：知识输出型合作方
- 方式：先科普，后植入品牌解决方案

**b. 问题解决型**
- 适合：实用技巧型合作方
- 方式：提出问题 → 分析原因 → 品牌方案

**c. 故事叙述型**
- 适合：情感共鸣型合作方
- 方式：用户故事/品牌故事

**d. 行业洞察型**
- 适合：观点输出型合作方
- 方式：行业分析 → 品牌实践

---

### 阶段5：内容撰写
**融合策略**：

**开头适配**：
- 采用合作方常用开头方式
- 引入品牌相关话题
- 保持合作方语气

**主体融合**：
- 使用合作方语言风格
- 采用合作方段落结构
- 植入品牌核心信息

**品牌信息呈现**：
- 严格遵循品牌档案
- 核心卖点完整呈现
- 数据准确引用

**结尾处理**：
- 参考合作方结尾习惯
- 加入品牌CTA（如有需要）

---

### 阶段6：风格适配检查
**检查清单**：

**开头**：
- [ ] 开头方式是否匹配合作方？
- [ ] 语气是否符合合作方习惯？

**语言**：
- [ ] 用词是否贴合合作方风格？
- [ ] 句式长度是否匹配？
- [ ] 表达习惯是否一致？

**结构**：
- [ ] 段落长度是否匹配？
- [ ] 段落数量是否相近？
- [ ] 分段习惯是否一致？

**格式**：
- [ ] emoji使用是否恰当？
- [ ] 特殊标签是否保留？
- [ ] 排版风格是否一致？

**价值**：
- [ ] 是否提供合作方受众期待的内容价值？
- [ ] 品牌信息是否自然融入？

---

### 阶段7：品牌信息核对
**与品牌档案逐项核对**：
- [ ] 核心卖点是否完整？
- [ ] 技术描述是否准确？
- [ ] 数据引用是否正确？
- [ ] 品牌调性是否保持？
- [ ] 是否有违规内容？

---

### 阶段8：格式适配
**根据合作方平台要求调整**：
- 长度要求
- 排版格式
- 配图要求
- 特殊标签

---

### 阶段9：输出交付
**交付内容**：
- 符合合作方格式要求的内容
- 品牌信息位置备注
- 风格适配说明

---

## 风格融合示例

### 示例1：知识科普型合作方
**合作方风格**：
- 开头：引用权威研究
- 语言：专业但不晦涩
- 结构：长段落，逻辑清晰
- emoji：稀少，仅关键处使用

**适配文章**：
```
【开头】
正如斯坦福睡眠中心专家Andrew Hubermann所言："虽然我们每天都在睡觉，但90%的人其实并不懂它。"今天，我想和大家聊聊关于睡眠的真相。

【主体 - 融合科普+品牌】
睡眠不是简单的"关机"，而是一个精密的生理过程。在大脑深处，有一套叫类淋巴系统的"清洁工"，只有在深度睡眠时才会开始工作......[科普内容]

这就是为什么精确的睡眠监测如此重要。悟昕AI智能睡眠仪采用与医院PSG同源的EEG脑电监测技术，能精准捕捉你的深睡比例、REM周期等关键指标。[品牌信息]

【结尾】
睡眠健康是一场长期修行，希望今天的分享对你有帮助。
```

## 常见风格类型

### 1. 学术严谨型
**特点**：引用权威、逻辑严密、数据支撑
**适配要点**：
- 强化数据和引用
- 逻辑链条清晰
- 避免过度情感化

### 2. 轻松活泼型
**特点**：口语化、emoji丰富、互动性强
**适配要点**：
- 使用口语化表达
- 适度使用emoji
- 增加互动语句

### 3. 情感共鸣型
**特点**：故事化、情感化、代入感强
**适配要点**：
- 用故事引入
- 建立情感连接
- 温暖真诚的语气

### 4. 实用干货型
**特点**：清单体、技巧型、可操作性强
**适配要点**：
- 提供具体方法
- 清晰的步骤说明
- 实用价值优先

### 5. 行业观点型
**特点**：深度分析、独立观点、前瞻性
**适配要点**：
- 行业洞察深度
- 独特观点输出
- 趋势预判
