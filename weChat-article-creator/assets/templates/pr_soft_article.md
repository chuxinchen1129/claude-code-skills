# 公关软文模板

## 适用场景
- 第三方媒体发布的品牌软文
- 行业媒体、新闻媒体的商业报道
- 需要保持第三方客观口吻的品牌推广

## 参考样本
- sample_08_sleep_breakdown.md（科学拆解型软文）

## 核心原则
1. **严格遵循品牌信息**：不自由发挥，所有品牌点需有依据
2. **新闻性开头**：从行业热点/趋势/问题切入，不是硬广
3. **自然植入**：品牌信息作为解决方案自然出现
4. **第三方口吻**：保持客观，避免过度营销化语言

## 文章结构

### 1. 新闻性开头（5-10%）
**目的**：建立新闻价值，吸引目标读者

**三种常见切入方式**：

**a. 行业热点切入**
```
标题：《睡眠经济爆发：千亿市场的创新机会》

开头：
随着"健康中国2030"战略的深入推进，睡眠健康正成为新的消费热点。艾瑞咨询数据显示，2027年中国助眠市场规模预计达6586亿元。在消费升级和健康意识觉醒的双重驱动下，睡眠产业正迎来前所未有的发展机遇。
```

**b. 产品/技术突破切入**
```
标题：《脑机接口技术落地睡眠领域：医疗级监测家用化成为可能》

开头：
近日，一款搭载医疗级EEG脑电监测技术的智能睡眠仪引发行业关注。该产品将医院昂贵的PSG监测技术微缩进家用设备，标志着脑机接口技术在消费级市场的首次大规模应用。
```

**c. 社会问题/痛点切入**
```
标题：《中国3亿睡眠障碍人群的"失眠困局"：精准化诊疗成为破局关键》

开头：
"我昨晚又失眠了......"这样的抱怨在现代都市中越来越常见。据《2025中国睡眠健康调查报告》显示，中国成年人失眠发生率高达48%，超过3亿人存在睡眠障碍。然而，在庞大的需求背后，是诊疗资源的严重匮乏和科学睡眠管理的缺失。
```

**要点**：
- 第一段必须有新闻价值
- 引用权威数据或事件
- 快速建立与品牌的关联方向

---

### 2. 问题分析/趋势解读（25-30%）
**目的**：深入分析行业问题或趋势，为品牌出场做铺垫

**内容方向**：

**a. 行业问题深度分析**
- 现状描述（数据支撑）
- 问题根源分析
- 现有解决方案的局限
- 消费者痛点

**b. 技术趋势解读**
- 技术发展背景
- 技术突破的难点
- 技术应用的价值
- 行业标杆案例

**c. 市场趋势分析**
- 市场规模与增长
- 消费需求变化
- 竞争格局演变
- 未来发展方向

**写作要点**：
- 引用权威数据和研究
- 分析要有深度，不只是表面现象
- 为品牌解决方案做逻辑铺垫
- 保持客观分析立场

---

### 3. 自然过渡到品牌（5-10%）
**目的**：从行业分析平滑过渡到品牌介绍

**过渡方式**：

**a. 技术突破过渡**
```
在这样的行业背景下，一些创新企业正在探索新的路径。悟昕科技近日发布的AI智能睡眠仪，尝试将医疗级脑电监测技术引入家用场景，为睡眠健康管理提供新的可能。
```

**b. 解决方案过渡**
```
针对上述痛点，悟昕科技提出了"监测-干预-反馈"的闭环管理模式。该公司创始人吴筱杰博士表示："真正的睡眠管理不应只停留在'知'的层面，更要实现'管'的突破。"
```

**c. 创新案例过渡**
```
在这一领域，悟昕科技的实践颇具代表性。该品牌将医院PSG监测的核心技术——EEG脑电监测，首次成功微缩进家用设备，实现了医疗级准确率的家用化突破。
```

**要点**：
- 保持客观报道口吻
- 不过度渲染，点到为止
- 自然承接上文分析

---

### 4. 品牌解决方案介绍（30-35%）
**目的**：介绍品牌如何解决上述问题

**内容结构**：

**a. 品牌背景简介**（简洁）
- 创始人/团队背景
- 品牌定位
- 核心理念

**b. 产品/服务介绍**（核心）
- 技术原理（通俗化）
- 核心功能（2-3个）
- 独特优势
- 数据支撑（实验室数据、第三方认证）

**c. 应用案例/效果**
- 用户案例（可选）
- 实验室数据
- 临床验证（如有）

**写作要点**：
- 严格遵循品牌档案核心卖点
- 每个品牌点需有数据或案例支撑
- 避免夸大或未经证实的内容
- 保持客观报道风格

---

### 5. 行业价值/用户价值（15-20%）
**目的**：阐述品牌带来的行业意义或用户价值

**内容方向**：

**a. 行业价值**
- 填补了什么空白
- 推动了什么进步
- 树立了什么标杆

**b. 用户价值**
- 解决了什么核心痛点
- 提供了什么独特价值
- 改善了什么体验

**c. 专家/第三方评价**（可选）
- 行业专家观点
- 媒体评价
- 合作机构评价

---

### 6. 展望/结尾（5-10%）
**目的**：总结全文，给出展望

**内容方向**：
- 行业发展趋势展望
- 品牌未来规划
- 开放式结尾

---

## 品牌信息植入技巧

### 1. 数据化呈现
```
悟昕采用的CNN+RNN混合神经网络算法，能够每32秒动态输出一次睡眠分期结果。实验室数据显示，其分期准确率达到医疗级16电极设备的80%以上。
```

### 2. 对比化呈现
```
与传统智能手环基于心率估算不同，悟昕直接读取大脑电活动，就像把医院的睡眠监测带回了家。
```

### 3. 场景化呈现
```
每天早晨，用户都能获得一份详尽的睡眠构架分析：深度睡眠占比如何？REM睡眠是否充足？夜间真正醒了多少次？
```

### 4. 原理化呈现
```
深度睡眠时，脑脊液会大量涌入大脑，像高压水枪一样冲刷脑细胞，带走代谢垃圾。悟昕的EEG监测技术能精准捕捉这一过程。
```

- 《睡眠经济爆发：千亿市场的创新机会》
- 《脑机接口技术落地睡眠领域》
- 《中国3亿睡眠障碍人群的"失眠困局"》

### 问题解决式标题
- 《从"知"到"管"：睡眠管理的代际跨越》
- 《医疗级脑电监测：家用化成为可能》

### 行业分析式标题
- 《睡眠3.0时代：闭环管理成新趋势》
- 《精准化诊疗：破局睡眠管理的关键》

## 写作禁忌

1. **过度营销化语言**：
   ❌ "震惊！""必看！""史上最强！"
   ✅ 保持客观报道口吻

2. **夸大功效**：
   ❌ "根治失眠""100%有效""立竿见影"
   ✅ "平均缩短30%入睡时间（实验室数据）"

3. **贬低竞品**：
   ❌ "其他品牌都是垃圾"
   ✅ 客观对比技术路线差异

4. **未经证实的内容**：
   ❌ 添加未经品牌方确认的信息
   ✅ 严格遵循品牌档案和提供的信息

5. **硬广开头**：
   ❌ "悟昕科技隆重推出......"
   ✅ 从行业热点/问题切入

---

## 检查清单

### 品牌信息核对
- [ ] 所有品牌信息是否与品牌档案一致？
- [ ] 核心卖点是否完整呈现？
- [ ] 数据是否有来源依据？
- [ ] 是否有夸大或未经证实的内容？

### 新闻性检查
- [ ] 开头是否有新闻价值？
- [ ] 是否保持了第三方客观口吻？
- [ ] 品牌信息是否自然植入？
- [ ] 是否避免了硬广感？

### 内容质量
- [ ] 行业分析是否有深度？
- [ ] 逻辑结构是否连贯？
- [ ] 过渡是否自然？
- [ ] 结尾是否有价值？
