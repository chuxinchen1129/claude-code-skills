# 官方媒体发布模板

## 适用场景
- 品牌官方公众号发布
- 品牌官网发布
- 官方媒体账号发布

## 参考样本
- sample_06_brand_story.md（创始人品牌故事）

## 核心原则
1. **品牌视角**：第一人称或品牌官方视角
2. **调性一致**：严格遵循品牌调性要求
3. **信息准确**：所有信息需与品牌档案一致
4. **情感连接**：与用户建立情感共鸣

## 与公关软文的区别

| 维度 | 公关软文 | 官方媒体发布 |
|------|----------|--------------|
| 视角 | 第三方客观报道 | 第一人称/品牌视角 |
| 口吻 | 新闻报道风格 | 品牌官方风格 |
| 目标 | 媒体发布，建立行业认知 | 官方发声，直接触达用户 |
| 调性 | 客观中立 | 符合品牌调性 |
| 灵活度 | 严格遵循提供信息 | 可更多展现品牌个性 |

## 文章类型

### 类型一：产品介绍类
**目的**：详细介绍产品功能、优势、使用场景

**结构**：
```
标题：产品名称 + 核心卖点

1. 引入（痛点/场景）
   - 用户痛点描述
   - 使用场景呈现
   - 引出产品

2. 产品核心理念
   - 设计初衷
   - 解决什么问题
   - 与市场方案的差异

3. 产品功能详解
   - 功能1：技术原理 + 用户价值
   - 功能2：技术原理 + 用户价值
   - 功能3：技术原理 + 用户价值

4. 产品优势
   - 技术优势（数据支撑）
   - 体验优势（场景描述）
   - 对比优势（竞品对比）

5. 用户案例/反馈（如有）
   - 真实用户故事
   - 使用前后对比

6. 购买/试用信息
   - 价格
   - 渠道
   - 优惠活动
```

---

### 类型二：品牌故事类
**目的**：传递品牌理念、价值观、初心

**参考**：sample_06_brand_story.md

**结构**：
```
标题：品牌名称 + 故事主题

1. 开篇引入
   - 创始人第一人称
   - 或品牌故事开场

2. 创业初心
   - 为什么做这件事？
   - 遇到了什么问题？
   - 为什么是这个问题？

3. 探索过程
   - 如何寻找解决方案？
   - 经历了哪些挑战？
   - 关键转折点

4. 产品/品牌诞生
   - 灵感来源
   - 品牌命名故事
   - 核心理念

5. 品牌愿景
   - 想成为什么？
   - 想创造什么价值？
   - 对用户的承诺

6. 情感升华
   - 品牌与用户的连接
   - 未来展望
```

**写作要点**：
- 真诚第一，不要煽情过度
- 用具体故事代替抽象陈述
- 展现创始人的真实思考
- 自然传递品牌价值观

## 标题建议

### 官方发布标题特点
- 清晰直接，不过度标题党
- 体现品牌调性
- 包含核心信息

### 标题模板

**产品介绍类**：
- "悟昕AI智能睡眠仪：医疗级脑电监测，让好睡眠可以被看见"
- "从'知'到'管'：悟昕如何重新定义睡眠管理？"

**品牌故事类**：
- "为什么做悟昕？一封来自创始人的信"
- "悟昕的诞生：用科学的'物'，满足内心的'心'"

**活动公告类**：
- "悟昕科技发布AI智能睡眠仪，引领睡眠管理进入3.0时代"
- "悟昕科技与XX医院达成战略合作，共建睡眠健康诊疗新标准"

**技术解读类**：
- "EEG脑电监测：为什么它是睡眠监测的'金标准'？"
- "CES经颅微电流刺激：科学助眠的物理路径"

## 开头方式

### 1. 创始人第一人称
```
大家好，我是Leo，悟昕科技的创始人。也许你和我一样，也曾无数次渴望一场"大脑的静默"......
```

### 2. 品牌视角
```
在人的一生中，有三分之一的时间在睡觉。但90%的人其实并不懂它。今天，悟昕想和你聊聊关于睡眠的真相。
```

### 3. 用户痛点共鸣
```
你是否也经历过这样的夜晚：躺在床上辗转反侧，大脑却异常清醒......悟昕懂你的困扰，因为我们正在用科技的力量，为无数人找回好睡眠。
```

### 4. 场景化描述
```
清晨6点，闹钟响起。你睁开眼，感觉身体依然沉重......这不是你想要的醒来方式。悟昕相信，每个人都有权利享受一场高质量的睡眠。
```

## 品牌调性保持

### 悟昕科技调性要点
- **语气**：专业严谨但不冰冷，温暖真诚
- **表达**：科学循证，数据驱动
- **价值观**：用科学满足"心"的安宁
- **品牌理念**：禅意绿洲 (Zen Oasis)

### 语言风格
- 避免过度营销化语言
- 科学术语要通俗化解释
- 用数据支撑观点
- 保持真诚不煽情

### 禁忌表达
- ❌ "震惊全网""史上最强""不看后悔"
- ❌ "根治""100%有效""立竿见影"
- ❌ 与药物治疗效果直接对比
- ✅ 专业、真诚、有温度

## 情感连接技巧

### 1. 理解用户痛点
```
我们深知，每一个失眠的夜晚，都是身心的双重折磨。这就是为什么悟昕不只是提供一款产品，更希望成为你睡眠健康的伙伴。
```

### 2. 传递品牌温度
```
"悟昕"的命名，凝结了我们的初心——"悟"是对本质的深度思考，"昕"是破晓的希望之光。我们希望用科学的"物"，来满足你内心"心"的安宁。
```

### 3. 展现品牌担当
```
睡眠问题从来不是一个简单的生理问题，它关乎我们的整体生命质量。悟昕所做的，不仅是一款产品，更是一场关于睡眠健康的探索。
```

## 检查清单

### 品牌调性
- [ ] 语气是否符合品牌调性？
- [ ] 是否保持了真诚不煽情？
- [ ] 是否避免了过度营销化语言？

### 信息准确
- [ ] 所有信息是否与品牌档案一致？
- [ ] 核心卖点是否准确呈现？
- [ ] 数据是否有来源依据？

### 内容质量
- [ ] 逻辑结构是否清晰？
- [ ] 开头是否吸引人？
- [ ] 情感连接是否自然？
- [ ] CTA是否明确？

### 格式规范
- [ ] 是否符合品牌VI规范？
- [ ] 排版是否清晰美观？
- [ ] 图片风格是否统一？
