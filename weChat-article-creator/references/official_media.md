# 官方媒体发布指南

## 什么是官方媒体发布？

官方媒体发布是指品牌通过官方渠道（公众号、官网、官方自媒体账号等）直接向用户传达信息的内容形式。

## 核心特点

1. **品牌视角**：第一人称或品牌官方视角
2. **调性一致**：严格遵循品牌调性要求
3. **直接触达**：直接面向目标用户
4. **情感连接**：与用户建立情感共鸣
5. **信息准确**：所有信息需与品牌档案一致

## 与公关软文的区别

| 维度 | 公关软文 | 官方媒体发布 |
|------|----------|--------------|
| 发布渠道 | 第三方媒体/行业媒体 | 品牌官方渠道 |
| 视角 | 第三方客观报道 | 品牌第一人称/官方视角 |
| 语气 | 新闻报道风格 | 品牌官方风格 |
| 灵活度 | 严格遵循提供信息 | 可更多展现品牌个性 |
| 目标 | 建立行业认知/权威 | 直接触达用户/建立连接 |

## 品牌调性一致

### 悟昕科技调性要点

**语气**：专业严谨但不冰冷，温暖真诚

**表达**：科学循证，数据驱动

**价值观**：用科学满足"心"的安宁

**品牌理念**：禅意绿洲 (Zen Oasis)

---

### 语言风格保持

**使用**：
- 专业术语通俗化解释
- 数据和事实支撑观点
- 温暖真诚的语气
- 理性但有温度

**避免**：
- 过度营销化语言
- 夸张或绝对化表述
- 煽情过度
- 与品牌调性不符的表达

### 示例对比

| ❌ 不符合调性 | ✅ 符合悟昕调性 |
|--------------|----------------|
| "家人们！悟昕太牛了！赶紧冲！" | "悟昕相信，科学的力量可以为每个人带来更好的睡眠。" |
| "不看后悔！失眠的终极解决方案！" | "悟昕AI智能睡眠仪，用科学的方式，读懂你的睡眠。" |
| "买了绝对不会后悔！" | "我们希望悟昕能成为你睡眠健康的伙伴。" |

## 内容类型

### 类型一：产品介绍类

**目的**：详细介绍产品功能、优势、使用场景

**结构建议**：
```
标题：产品名称 + 核心卖点

1. 引入（痛点/场景）
   - 用户痛点描述
   - 使用场景呈现
   - 引出产品

2. 产品核心理念
   - 设计初衷
   - 解决什么问题
   - 与市场方案的差异

3. 产品功能详解
   - 功能1：技术原理 + 用户价值
   - 功能2：技术原理 + 用户价值
   - 功能3：技术原理 + 用户价值

4. 产品优势
   - 技术优势（数据支撑）
   - 体验优势（场景描述）
   - 对比优势

5. 用户案例/反馈（如有）

6. 购买/试用信息
```

---

### 类型二：品牌故事类

**目的**：传递品牌理念、价值观、初心

**参考**：sample_06_brand_story.md

**结构建议**：
```
标题：品牌名称 + 故事主题

1. 开篇引入
   - 创始人第一人称
   - 或品牌故事开场

2. 创业初心
   - 为什么做这件事？
   - 遇到了什么问题？
   - 为什么是这个问题？

3. 探索过程
   - 如何寻找解决方案？
   - 经历了哪些挑战？
   - 关键转折点

4. 产品/品牌诞生
   - 灵感来源
   - 品牌命名故事
   - 核心理念

5. 品牌愿景
   - 想成为什么？
   - 想创造什么价值？
   - 对用户的承诺

6. 情感升华
   - 品牌与用户的连接
   - 未来展望
```

**写作要点**：
- 真诚第一，不要煽情过度
- 用具体故事代替抽象陈述
- 展现创始人的真实思考
- 自然传递品牌价值观

---

### 类型三：活动公告类

**目的**：宣布品牌活动、合作、里程碑等

**结构建议**：
```
标题：活动名称 + 核心信息

1. 活动宣布
   - 什么事

   - 为什么重要

2. 活动详情
   - 活动内容
   - 参与方式
   - 活动亮点

3. 品牌意义
   - 对品牌的意义
   - 对用户的价值
   - 行业影响

4. 行动号召
   - 如何参与
   - 联系方式
   - 截止时间
```

---

### 类型四：技术解读类

**目的**：深度解读产品技术原理，建立专业形象

**结构建议**：
```
标题：技术名称 + 价值主张

1. 技术引入
   - 为什么需要这项技术？
   - 这项技术解决什么问题？

2. 技术原理解析
   - 技术原理（通俗化解释）
   - 技术难点
   - 突破性创新

3. 技术优势
   - 与传统方案对比
   - 数据支撑
   - 第三方认证

4. 应用场景
   - 具体应用案例
   - 用户价值体现

5. 未来展望
   - 技术演进方向
   - 更多应用可能
```

## 开头方式

### 1. 创始人第一人称

**适用**：品牌故事、重要公告

```
大家好，我是Leo，悟昕科技的创始人。也许你和我一样，也曾无数次渴望一场"大脑的静默"......
```

**要点**：
- 真诚、亲切
- 避免过度煽情
- 快速进入主题

---

### 2. 品牌视角

**适用**：产品介绍、科普教育

```
在人的一生中，有三分之一的时间在睡觉。但90%的人其实并不懂它。今天，悟昕想和你聊聊关于睡眠的真相。
```

**要点**：
- 专业但不冰冷
- 提供价值导向
- 符合品牌调性

---

### 3. 用户痛点共鸣

**适用**：产品介绍、解决方案

```
你是否也经历过这样的夜晚：躺在床上辗转反侧，大脑却异常清醒......悟昕懂你的困扰，因为我们正在用科技的力量，为无数人找回好睡眠。
```

**要点**：
- 真实场景描述
- 建立情感连接
- 自然引出品牌

---

### 4. 场景化描述

**适用**：产品介绍、使用指南

```
清晨6点，闹钟响起。你睁开眼，感觉身体依然沉重......这不是你想要的醒来方式。悟昕相信，每个人都有权利享受一场高质量的睡眠。
```

**要点**：
- 具体场景
- 引发共鸣
- 传递品牌温度

---

## 情感连接技巧

### 1. 理解用户痛点

**原则**：展现品牌对用户痛点的理解，建立共鸣

```
我们深知，每一个失眠的夜晚，都是身心的双重折磨。这就是为什么悟昕不只是提供一款产品，更希望成为你睡眠健康的伙伴。
```

**要点**：
- 真诚理解，不敷衍
- 从用户角度出发
- 传递品牌关怀

---

### 2. 传递品牌温度

**原则**：通过品牌故事、理念传递温度

```
"悟昕"的命名，凝结了我们的初心——"悟"是对本质的深度思考，"昕"是破晓的希望之光。我们希望用科学的"物"，来满足你内心"心"的安宁。
```

**要点**：
- 品牌命名/理念自然融入
- 展现品牌价值观
- 避免"说教感"

---

### 3. 展现品牌担当

**原则**：展现品牌的社会责任和行业担当

```
睡眠问题从来不是一个简单的生理问题，它关乎我们的整体生命质量。悟昕所做的，不仅是一款产品，更是一场关于睡眠健康的探索。
```

**要点**：
- 超越产品本身
- 连接更大价值
- 不空洞喊口号

## 标题建议

### 官方发布标题特点
- 清晰直接，不过度标题党
- 体现品牌调性
- 包含核心信息

### 标题模板

**产品介绍类**：
- "悟昕AI智能睡眠仪：医疗级脑电监测，让好睡眠可以被看见"
- "从'知'到'管'：悟昕如何重新定义睡眠管理？"

**品牌故事类**：
- "为什么做悟昕？一封来自创始人的信"
- "悟昕的诞生：用科学的'物'，满足内心的'心'"

**活动公告类**：
- "悟昕科技发布AI智能睡眠仪，引领睡眠管理进入3.0时代"
- "悟昕科技与XX医院达成战略合作，共建睡眠健康诊疗新标准"

**技术解读类**：
- "EEG脑电监测：为什么它是睡眠监测的'金标准'？"
- "CES经颅微电流刺激：科学助眠的物理路径"

## 常见误区

### 1. 过度煽情
**表现**：大量感叹词、情绪化表达

**避免方法**：
- 真诚第一，不过度渲染
- 用事实和故事说话
- 保持专业底色

---

### 2. 硬广感太强
**表现**：全文推销产品，缺乏内容价值

**避免方法**：
- 提供有价值的信息
- 故事化表达
- 建立情感连接

---

### 3. 调性不一致
**表现**：语气、用词与品牌调性不符

**避免方法**：
- 参考品牌档案调性要求
- 保持专业但有温度
- 避免极端化表达

---

### 4. 信息不准确
**表现**：与品牌档案信息不一致

**避免方法**：
- 严格遵循品牌档案
- 核心信息需品牌方确认
- 数据标注来源

---

### 5. 缺乏情感连接
**表现**：内容过于冰冷，像产品说明书

**避免方法**：
- 用户视角出发
- 真诚理解用户痛点
- 传递品牌温度
