# 风格匹配指南

## 如何分析范文

### 第一步：通读感知

**感受维度**：
- 语气：亲切/严肃/幽默/冷静？
- 节奏：快/慢/变化？
- 情感：热烈/克制/温暖/理性？
- 距离感：像朋友/像老师/像专家？

**用户样本风格参考**：
- 参见 `assets/samples/` 目录中的4篇样本文章
- 参见 `assets/user_style_analysis.md` 获取完整的风格画像
- 主要特征：零售行业商业分析、数据驱动、故事化叙述、专业洞察

### 第二步：特征提取

#### 1. 句式特征

**短句型风格**：
```
特点：句子简短有力
示例："想通了。也就这样。"
适用：情感共鸣、强调观点
```

**长句型风格**：
```
特点：句子较长，信息密度高
示例："在这个过程中，我们需要不断地反思、调整、优化，才能找到最适合自己的节奏"
适用：专业分析、深度思考
```

**问句型风格**：
```
特点：频繁使用设问、反问
示例："你有没有想过？为什么..."
适用：引发思考、互动感强
```

#### 2. 词汇特征

**口语化标记**：
- 叹词：哎、嘛、呗、哦
- 程度词：超级、特别、简直
- 时间词：那会儿、那会儿、现在

**书面化标记**：
- 成语使用频率
- 专业术语密度
- 抽象词汇比例

**特色词汇清单**：
- 常用形容词（如："治愈"、"扎心"、"上头"）
- 常用动词（如："解锁"、"get"、"拿捏"）
- 常用句式（如："不得不说..."、"emmm..."）

#### 3. 语气特征

**温暖型**：
- 多用鼓励性语言
- 使用"我们一起"、"你值得"
- 情感表达直白

**冷静型**：
- 客观陈述为主
- 少用感叹号
- 理性分析

**幽默型**：
- 自嘲或调侃
- 意想不到的比喻
- 反转句式

**犀利型**：
- 直击痛点
- 不拐弯抹角
- 观点鲜明

#### 4. 节奏特征

**快节奏**：
- 段落短小
- 句子简短
- 信息密集

**慢节奏**：
- 段落较长
- 描述细致
- 铺垫充分

**变化节奏**：
- 快慢结合
- 重要内容放慢
- 次要内容加快

### 第三步：风格画像

将以上特征整合成一个风格画像：

```
风格名称：[如：温暖治愈型朋友]

核心特征：
- 句式：多用短句，偶尔长句解释
- 词汇：口语化，常用"呀"、"呢"
- 语气：温暖、鼓励、像朋友聊天
- 节奏：整体舒缓，关键处放慢
- 特色：喜欢用"～"、"❤️"等符号

适用场景：情感类、成长类文章
```

## 去 AI 味技巧清单

### 1. 避免过度结构化

❌ AI 味：
"首先...其次...最后..."
"第一...第二...第三..."
"一方面...另一方面..."

✅ 人味：
"更重要的是..."
"但很多人忽略了一个点..."
"说个你可能不知道的事..."

### 2. 减少"完美对称"

❌ AI 味：
"不仅提高了效率，还降低了成本"
"既保证了质量，又节省了时间"

✅ 人味：
"效率提高了很多，成本自然就下来了"
"质量没打折，时间还省了"

### 3. 加入具体细节

❌ AI 味：
"坚持学习很重要"

✅ 人味：
"去年这个时候，我也是每天下班后刷手机，直到有一天..."

### 4. 使用口语化表达

❌ AI 味：
"因此，我们应该采取相应的措施"

✅ 人味：
"所以啊，真得改改了"
"说白了就是..."
"老实讲..."

### 5. 打破"完美逻辑"

❌ AI 味：
每个观点都有完美论证，逻辑严密

✅ 人味：
"说实话，我也不太确定..."
"这可能因人而异"
"至少对我来说是这样"

### 6. 加入个人情绪

❌ AI 味：
客观中立，无情感色彩

✅ 人味：
"看到这个数据，我惊了"
"有时候真的挺无奈的"
"这种感觉太爽了"

### 7. 不确定性的表达

❌ AI 味：
"一定能..."、"必然..."、"必定..."

✅ 人味：
"应该可以..."、"试试看..."、"可能会..."

### 8. 使用"你"和"我"

❌ AI 味：
"人们应该..."、"读者需要..."

✅ 人味：
"你是不是也有这样的经历..."
"我试过..."
"咱们一起..."

## 风格复现方法

### 第一步：风格模仿清单

根据范文分析，列出模仿要点：

```
□ 句式：[如：多用短句，每段不超过3行]
□ 词汇：[如：用"咱"代替"我们"]
□ 语气：[如：像和朋友聊天]
□ 节奏：[如：重要内容单独一段]
□ 符号：[如：适当用～、❤️]
□ 特色：[如：开头常用"嘿，你..." ]
```

### 第二步：逐句对照

在写作时：
1. 写完后，逐句对照风格清单
2. 检查是否符合风格特征
3. 不符合的地方重新表述

### 第三步：朗读检验

把文章读出来：
- 是否自然？
- 是否像范文那样？
- 哪里不像？为什么？

## 常见风格类型

### 1. 职场导师型
```
特征：专业、理性、有说服力
句式：论证清晰，逻辑严密
词汇：专业但不晦涩
语气：权威但亲和
```

### 2. 暖心朋友型
```
特征：温暖、共情、治愈
句式：短句多，节奏舒缓
词汇：口语化，语气词多
语气：像闺蜜聊天
```

### 3. 幽默段子手型
```
特征：轻松、有趣、反转
句式：意想不到的转折
词汇：网络用语，梗多
语气：调侃、自嘲
```

### 4. 理性分析型
```
特征：客观、深度、数据驱动
句式：完整、严谨
词汇：专业术语准确
语气：冷静、克制
```

### 5. 故事讲述型
```
特征：画面感、代入感强
句式：描述细致，场景化
词汇：具体、形象
语气：娓娓道来
```
