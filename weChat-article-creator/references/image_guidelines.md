## 🎯 配图操作流程（5 步）⭐

### 第 1 步：确定配图需求

**明确清单**：
- [ ] 需要几张图片？
- [ ] 每张图片的用途是什么？
- [ ] 图片类型是什么？（Unsplash/品牌素材/数据图表）
- [ ] 优先级如何？（高/中/低）

**配图数量参考**：
- 文章 < 1000 字：2-3 张
- 文章 1000-2000 字：4-6 张
- 文章 2000-3000 字：6-8 张
- 文章 > 3000 字：每 500 字配 1 张

**配图位置规划**：
1. **开头封面**：必须有（高优先级）
2. **每隔 3-5 段**：1 张（中优先级）
3. **关键段落**：重点配图（高优先级）
4. **结尾**：可选（低优先级）

**示例**：

```
【配图需求清单】

**配图数量**：5 张
**图片类型**：Unsplash 免费图片
**文章长度**：2500 字

**图片 1**：文章开头封面（高优先级）
- 用途：吸引注意力，营造睡眠氛围
- 关键词：sleep, night, peaceful bedroom

**图片 2**：睡眠周期图示（高优先级）
- 用途：解释睡眠科学原理
- 关键词：sleep cycle, circadian rhythm

**图片 3**：睡眠问题场景（中优先级）
- 用途：引发读者共鸣
- 关键词：insomnia, sleep problems

**图片 4**：解决方案示意（中优先级）
- 用途：展示解决方案
- 关键词：technology, health monitoring

**图片 5**：健康睡眠场景（低优先级）
- 用途：展示改善后的效果
- 关键词：good sleep, happy morning
```

### 第 2 步：选择图片来源

**图片来源优先级**（4 个层次）：

**Tier 1：品牌素材**（最高优先级）
- ✅ 品牌官方提供的图片
- ✅ 用户自己拍摄的图片
- ✅ 定制设计的数据图表

**使用场景**：
- 品牌推广文章
- 产品介绍
- 官方媒体发布

**Tier 2：Unsplash**（高优先级）
- ✅ 完全免费，可商用
- ✅ 高质量，专业摄影师拍摄
- ✅ 图片多样，覆盖广泛主题

**使用场景**：
- 自媒体文章
- 科普文章
- 商业分析

**Tier 3：Pexels/Pixabay**（中优先级）
- ✅ 免费可商用
- ✅ 支持中文搜索（Pexels）
- ✅ 图片数量多

**使用场景**：
- Unsplash 没有合适的图片时
- 需要中文搜索时

**Tier 4：其他来源**（低优先级，需验证）
- ⚠️ Google 图片搜索（多数有版权）
- ⚠️ 百度图片（版权不清晰）

**使用场景**：
- 前三个来源都没有合适的图片
- 必须逐个确认版权

**示例**：

```
【图片来源选择】

图片 1：Unsplash（品牌无合适素材）
图片 2：品牌素材（品牌提供的数据图表）
图片 3：Unsplash
图片 4：品牌素材（产品图片）
图片 5：Unsplash
```

   - "working happy"（工作愉快）

   - close up（特写）


### 第 5 步：插入图片到文章

**创建图片文件夹**：
```bash
mkdir -p assets/output/images/[文章名]/
```

**按顺序命名**：
```
image-01.jpg
image-02.jpg
image-03.jpg
...
```

**在文章中插入**：
```html
<!-- 开头封面图 -->
<figure>
  <img src="images/image-01.jpg" alt="宁静的睡眠环境">
  <figcaption>良好的睡眠环境是健康睡眠的基础</figcaption>
</figure>

<!-- 正文配图 -->
<p>...段落内容...</p>

<figure>
  <img src="images/image-02.jpg" alt="睡眠周期示意图">
  <figcaption>图1：人体睡眠周期示意图</figcaption>
</figure>

<p>...继续正文...</p>
```

**验证显示效果**：
1. **本地预览**：在浏览器中打开 HTML，检查图片显示
2. **移动端测试**：在手机上查看，确保图片自适应
3. **加载速度**：检查图片加载速度（如果过慢，需要压缩）

**完成检查**：

## 常见问题

### Q1: 如果 Unsplash 没有合适的图片怎么办？

**A**:
1. 尝试不同的关键词组合
2. 使用 Pexels 或 Pixabay
3. 使用品牌素材（如有）
4. 考虑是否真的需要这张图

### Q2: 图片可以修改吗？

**A**:
- ✅ 可以裁剪、调整亮度、压缩大小
- ✅ 可以添加滤镜（保持风格一致）
- ⚠️ 不要修改图片含义
- ⚠️ 不要遮挡主要元素

### Q3: 图片大小如何优化？

**A**:
- **目标**：500KB - 3MB
- **方法**：
  - 使用在线工具：TinyPNG（https://tinypng.com）
  - 或 Photoshop：存储为 Web 格式
  - 或脚本批量压缩

### Q4: 配图需要说明文字吗？

**A**:
- **必须有**：alt 属性（为了 SEO 和无障碍访问）
- **可选**：figcaption（图片说明）
- **原则**：说明文字应该补充或解释图片，不要只是重复

### Q5: 如何确保图片风格一致？

**A**:
- ✅ 使用相似的关键词（风格限定词）
- ✅ 保持色调一致（都温暖、都冷调）
- ✅ 保持构图方式相似（都是特写、都是广角）
- ✅ 避免混用截然不同的风格
