## 🎯 核心原则：选题讨论必做 ⭐⭐⭐

### 必做原则

**绝对禁止**：直接开始写文章！

**必须执行**：
1. ✅ **先讨论选题**：提供 3-4 个选题方向
2. ✅ **等待用户选择**：用户确认后再开始撰写
3. ✅ **明确每个选题**：标题、角度、工作量、优劣势、大纲
4. ✅ **避免方向错误**：选题不对，努力白费

### 为什么必须讨论选题？

**避免重写**：
- ❌ 不讨论就写 → 用户不满意 → 全部重写（浪费 2-3 小时）
- ✅ 先讨论选题 → 用户确认 → 一次写对（节省 80% 时间）

**提升质量**：
- ✅ 明确写作角度（避免跑题）
- ✅ 控制工作量（避免过大或过小）
- ✅ 匹配用户需求（避免方向偏差）

**真实案例**：

❌ **不讨论的错误示例**：
> 用户："写一篇关于 Claude Code 的文章"
> AI：直接开始写"Claude Code 是 Anthropic 公司推出的 AI 编程工具..."
> 用户："不对！我想写的是它的使用心得，不是产品介绍"
> 结果：全部重写

✅ **先讨论的正确示例**：
> 用户："写一篇关于 Claude Code 的文章"
> AI："我准备了 3 个选题方向，你看哪个更符合你的想法：
>
> **选题1**：产品介绍（适合读者不了解 Claude Code 的情况）
> **选题2**：使用心得（适合分享实际使用经验）
> **选题3**：竞品对比（适合对比分析）
>
> 用户："选题2，使用心得"
> 结果：一次写对

## 🚀 不同内容类型的选题策略

### 商业分析文章

**选题方向**：
1. **企业成功案例**：分析某企业的成功原因
2. **行业趋势分析**：解读行业发展趋势
3. **商业模式解读**：分析某商业模式的特点
4. **竞品对比分析**：对比多个产品/品牌

**示例选题**：
- 选题1：Whole Foods 为什么能成功？
- 选题2：有机食品行业趋势分析
- 选题3：Whole Foods vs ALDI 商业模式对比

### 科普文章

**选题方向**：
1. **科学拆解型**：深入解释科学原理
2. **科普清单型**：列举科学知识点
3. **创始人故事型**：通过人物故事讲科学

**示例选题**：
- 选题1：相对论的科学原理（科学拆解）
- 选题2：关于相对论的 5 个真相（科普清单）
- 选题3：爱因斯坦是如何发现相对论的（创始人故事）

### 品牌推广文章

**选题方向**：
1. **产品介绍**：介绍产品功能和特点
2. **品牌故事**：讲述品牌发展历程
3. **行业洞察**：从行业角度引出品牌
4. **用户案例**：展示真实使用效果

**示例选题**（悟昕科技）：
- 选题1：悟昕 AI 睡眠管理系统介绍
- 选题2：悟昕品牌故事：为什么我们关注睡眠
- 选题3：睡眠行业趋势 + 悟昕解决方案
- 选题4：用户案例：如何改善睡眠质量

## 💡 选题技巧

### 技巧 1：结合用户需求

**问题清单**：
- [ ] 文章目标是什么？（科普/推广/分析）
- [ ] 目标读者是谁？（专业人士/普通读者）
- [ ] 读者痛点是什么？
- [ ] 想要传递什么核心信息？

**示例**：
- 目标：推广悟昕睡眠产品
- 读者：睡眠不好的中年人
- 痛点：失眠、睡眠质量差
- 核心信息：悟昕能改善睡眠

### 技巧 2：控制工作量

**适中原则**：
- ✅ 2000-3000 字（自媒体）
- ✅ 3-5 个主要部分
- ✅ 每部分 2-3 个小节

**避免**：
- ❌ 过大：试图覆盖所有内容
- ❌ 过小：内容不够深入

### 技巧 3：差异化角度

**思考维度**：
- 时间维度：过去/现在/未来
- 角度维度：用户/企业/行业
- 深度维度：入门/进阶/专家

**示例（Claude Code）**：
- 时间："Claude Code 2024 年发布的新功能"
- 角度："开发者视角下的 Claude Code"
- 深度："Claude Code 高级使用技巧"

### 技巧 4：热点结合

**方法**：
1. 查看当前热点（小红书、微信、微博）
2. 找到与主题的结合点
3. 自然融入（不要硬蹭）

**示例**：
- 热点："AI 编程工具竞争加剧"
- 结合："2024 年 AI 编程工具大对比，Claude Code 脱颖而出"

## ✅ 选题质量检查

### 合格标准

**必须满足**：
- [ ] 至少提供 3 个选题选项
- [ ] 每个选题包含：标题、角度、工作量、优劣势、大纲

- [ ] 选题符合用户需求

### 优秀标准

**加分项**：
- [ ] 选题有差异化（不重复）
- [ ] 有明确的读者定位
- [ ] 考虑了时效性和热点
- [ ] 大纲合理、逻辑清晰

### 不合格标准

**必须避免**：
- ❌ 只提供 1 个选题

- ❌ 不等待用户选择就直接开始写
- ❌ 选题偏离用户需求
- ❌ 工作量过大或过小

## ❓ 常见问题

### Q1: 选题讨论需要多长时间？

**A**:
- **快速讨论**：5-10 分钟（用户需求明确）
- **标准讨论**：10-15 分钟（需要提供多个选题）
- **深度讨论**：15-30 分钟（需要调研后确定选题）

### Q2: 如果用户对选题都不满意怎么办？

**A**:
1. 询问用户的具体想法
2. 提供更多选题选项
3. 根据用户反馈调整选题角度
4. 必要时重新调研

### Q3: 可以只提供 1 个选题吗？

**A**:
- ❌ **不推荐**：只提供 1 个选题 = 没有选择
- ✅ **建议**：至少 3 个选题，让用户有选择空间
- ✅ **最佳**：3-4 个选题（太多会让用户选择困难）

### Q4: 选题讨论后，用户又改变想法了怎么办？

**A**:
- 先确认是否真的需要改变方向
- 如果用户坚持改变 → 重新讨论选题
- 如果只是小调整 → 在原选题基础上微调
- 记录用户偏好，下次参考

### Q5: 如何确保选题质量？

**A**:
- ✅ 选题符合用户需求
- ✅ 工作量适中（2000-3000 字）
- ✅ 有明确的角度和定位
- ✅ 大纲合理、逻辑清晰
- ✅ 考虑了时效性和热点

## 原有内容：爆款标题的 10 大特征

### 1. 数字承诺
- "我用了这3个方法，月入从5k到3w"
- "30天，我彻底改变了..."

### 2. 强烈对比
- "普通人也能做到的xx"
- "从小白到大神，我只用了一周"

### 3. 痛点直击
- "你是不是也经常xx？"
- "为什么你总是xx？"

### 4. 悬念设置
- "没想到，竟然是因为..."
- "这个秘密，90%的人都不知道"

### 5. 权威背书
- "清华北大都在用..."
- " according to xx研究..."

### 6. 紧迫感
- "再不xx就晚了"
- "今年最后的xx机会"

### 7. 独家揭秘
- "终于有人说出来了"
- "行业内幕大公开"

### 8. 实用价值
- "建议收藏！xx全攻略"
- "这一篇就够了"

### 9. 情感共鸣
- "看完想哭..."
- "这就是我的真实经历"

### 10. 热点借势
- 结合当前热点事件
- 使用流行词汇

## 小红书/微信热点库（需持续更新）

### 当前常见热梗
- "反向消费"：理性消费、不买贵的只买对的
