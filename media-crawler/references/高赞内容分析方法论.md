## 🎯 核心方法论

### 第一步：获客价值分计算

#### 获客价值分公式

```
获客价值分 = 点赞数 × 1 + 收藏数 × 2 + 评论数 × 3 + 分享数 × 4
```

#### 权重逻辑说明

| 互动类型 | 权重 | 含义 | 为什么重要 |
|---------|------|------|-----------|
| 点赞 | 1 | 一般关注 | 表示用户看到并认可，但互动深度浅 |
| 收藏 | 2 | 深度兴趣 ⭐ | 表示用户有强烈意向，会重复查看 |
| 评论 | 3 | 互动意愿 | 表示用户愿意表达，可能咨询 |
| 分享 | 4 | 传播价值 | 表示用户愿意推荐，传播价值最高 |

**关键洞察**：
- **收藏是核心指标**：收藏代表强烈意向，权重点高
- **互动比曝光重要**：1000个收藏比1万个点赞更有价值
- **评论代表商机**：评论的用户更有可能咨询转化

#### 实施步骤

```python
import pandas as pd

def calculate_customer_value_score(df):
    """
    计算获客价值分

    Args:
        df: 包含点赞、收藏、评论、分享数据的DataFrame

    Returns:
        df: 添加了'获客价值分'列的DataFrame
    """
    # 确保数值列是数值类型
    numeric_columns = ['点赞数', '收藏数', '评论数', '分享数']
    for col in numeric_columns:
        if col in df.columns:
            df[col] = pd.to_numeric(df[col], errors='coerce').fillna(0)

    # 计算获客价值分
    df['获客价值分'] = (
        df.get('点赞数', 0) * 1 +
        df.get('收藏数', 0) * 2 +
        df.get('评论数', 0) * 3 +
        df.get('分享数', 0) * 4
    )

    # 按获客价值分降序排列
    df = df.sort_values('获客价值分', ascending=False)

    return df

# 使用示例
df = pd.read_excel('原始数据.xlsx')
df_with_score = calculate_customer_value_score(df)
df_with_score.to_excel('含获客价值分的数据.xlsx', index=False)
```

### 第二步：爆款要素提取

#### 1. 标题要素分析

**五大标题要素**：

| 要素 | 说明 | 示例 |
|------|------|------|
| 具体数字 | 精确的数字，更有说服力 | "2000万债务"、"15年经验"、"95%成功率" |
| 痛点词汇 | 击中用户痛点的词 | "破产"、"负债"、"失眠"、"脱发" |
| 疑问句式 | 引发思考和点击 | "怎么办？"、"为什么？"、"如何？" |
| 情绪词汇 | 触发情绪反应 | "深夜"、"绝望"、"紧急"、"天啊" |
| 身份标签 | 建立专业权威 | "律师"、"医生"、"专家"、"导师" |

**爆款标题公式**：

```
公式1：痛点 + 数字 + 疑问
示例："负债200万怎么办？律师教你3招合法应对"

公式2：身份 + 数字 + 结果
示例："从医15年，我总结了5个失眠解决方案"

公式3：情绪 + 痛点 + 身份
示例："深夜破产的绝望，律师为你点亮希望"

公式4：数字 + 痛点 + 解决方案
示例："3个信号，告诉你该看律师了"
```

#### 2. 内容要素分析

**三段式内容结构**：

```
第一段：痛点描述（20%）
- 开头3秒抓注意（痛点/数据/身份）
- 描述用户的具体困境
- 引发共鸣

第二段：解决方案（60%）
- 提供2-3个具体方法
- 每个方法配真实案例（脱敏）
- 通俗易懂，避免专业术语

第三段：行动建议（20%）
- 引导用户关注
- 提供咨询入口
- 添加免责声明
```

**内容必含要素**：

- ✅ 开头3秒抓注意（痛点/数据/身份）
- ✅ 三段式结构（问题→方案→行动）
- ✅ 真实案例佐证（脱敏处理）
- ✅ 通俗化表达（避免专业术语）
- ✅ 免责声明（不构成法律/医疗意见）

#### 3. 封面要素分析

**封面设计三要素**：

| 要素 | 说明 | 示例 |
|------|------|------|
| 关键数字/结论 | 醒目展示 | "2000万"、"3招解决" |
| 专业身份标识 | 建立信任 | "XX律师"、"从业15年" |
| 对比强烈配色 | 吸引注意 | 蓝白、红黄对比色 |

**封面设计原则**：
- 简洁明了（不超过10个字）
- 关键信息大字号
- 专业配色（蓝/绿/红/黄）
- 添加个人照片或专业形象

**初期推荐**（第1-4周）：
- **实操指南**（50%）：具体可操作，转化率高
- **警示避坑**（30%）：建立信任，易传播

**中期推荐**（第5-8周）：
- **知识科普**（40%）：建立专业权威
- **实操指南**（40%）：保持高转化

**长期策略**（持续）：
- 根据数据分析调整比例
- 优先制作互动量高的类型
- 保持多样性，避免单一
