## 使用方式

### 作为风格学习参考

当需要生成某类内容时，参考对应样本的：
- **结构框架**：文章整体组织方式
- **语言风格**：语气、用词、句式
- **内容密度**：信息呈现节奏
- **互动设计**：如何引导读者

### 作为风格检查标准

生成内容后，对照样本检查：
- **结构一致性**：是否采用相似框架
- **风格匹配度**：语气、用词是否一致
- **质量标准**：是否达到样本水准

## 风格特征提取

### sample_06_brand_story.md - 品牌故事

**风格特征**：
```yaml
视角: 创始人第一人称
语气: 真诚、反思、使命感
结构:
  - 开篇：个人经历引入
  - 中间：问题分析+解决方案
  - 结尾：愿景+价值观
关键词: 创始人、初心、使命、科学、严谨
```

**核心写作手法**：
- 用"我"的个人故事建立信任
- 提出行业问题（割裂、不科学）
- 给出解决方案（既要、又要的产品理念）
- 传递品牌价值观（科学严谨+日常舒适）

### sample_07_sleep_science.md - 睡眠科普

**风格特征**：
```yaml
视角: 客观科普作者
语气: 轻松、有趣、专业
结构:
  - 开篇：引人共鸣的问题
  - 主体：10个科学知识点（清单式）
  - 结尾：产品价值+CTA
特色:
  - 大量数字和科学名词
  - 冷知识标签增加趣味
  - emoji点缀但不过度
```

**核心写作手法**：
- 开篇引发共鸣（"你是不是也有过..."）
- 清单式结构（10个知识点）
- 每个知识点：现象+原理+冷知识
- 软性植入产品价值
- CTA：搜"XXX"试试看

### sample_08_sleep_breakdown.md - 问题拆解

**风格特征**：
```yaml
视角: 科普分析者
语气: 专业、深入、理性
结构:
  - 开篇：问题提出（"失眠"万能词）
  - 主体：6种类型分类解释
  - 结尾：解决方案+产品介绍
特色:
  - 分类清晰（01/02/03...）
  - 专业术语+通俗解释
  - 强调复杂性（双向关系、多学科）
```

**核心写作手法**：
- 问题引入：质疑日常认知
- 分类拆解：6种不同类型
- 强调复杂性：症状与病因互为因果
- 产品定位：弥合鸿沟的解决方案

### sample_01_whole_foods.md - 商业分析

**风格特征**：
```yaml
视角: 商业分析师
语气: 客观、数据驱动、洞见
结构:
  - 开篇：奇迹品类描述
  - 主体：公司发展历程+成功要素
  - 结尾：行业启示
特色:
  - 强调数据（10%复合增长率、800万周客）
  - 对比鲜明（价格高40%但仍受欢迎）
  - 定位清晰（食品行业的苹果公司）
```
