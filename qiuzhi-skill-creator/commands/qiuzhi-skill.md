# 求职 Skill Creator

创建求职相关的 Claude Skills，通过交互式 SOP 流程引导你完成技能设计。

## 使用方法

直接告诉我想创建什么样的求职相关 Skill，例如：
- "帮我创建一个简历优化 Skill"
- "我想做一个面试准备助手"
- "创建职位搜索辅助工具"

然后我会通过 4 个阶段的交互式流程，帮你构建一个完整的 Skill。
