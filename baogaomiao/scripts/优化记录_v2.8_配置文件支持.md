#### 2.2 修改 `__init__` 方法
```python
def __init__(self, output_dir: Optional[Path] = None):
    # 加载配置
    config = self._load_config()

    # 应用配置到实例变量
    self.MAX_CHINESE_CHARS_PER_LINE = int(config['max_chinese_chars_per_line'])
    self.MAX_ENGLISH_LENGTH = int(config['max_english_length'])
    self.MAX_HIGHLIGHT_TITLE_LENGTH = int(config['max_highlight_title_length'])
    self.MAX_SUMMARY_TEXT_LENGTH = int(config['max_summary_text_length'])
    self.container_width = int(config['container_width'])
    self.container_height = int(config['container_height'])
    self.page_num_font_scale = float(config['page_num_font_scale'])
    self.title_font_scale = float(config['title_font_scale'])
```

#### 2.3 修改 `generate_cover` 方法
```python
# 计算动态值
page_num_font_size = f"{3.35 * self.page_num_font_scale:.2f}"
```

## 修复的问题

### 问题1: YAML 格式错误
**错误**: `page_num_font_scale: 2 0` 被解析为列表 `[2, 0]`
**修复**: 改为 `page_num_font_scale: 2.0`

### 问题2: 输出目录路径错误
**错误**: 输出到 `~/.claude/skills/...` 而非 DMS 目录
**修复**: 更新 `default_output_dir` 为 `~/Desktop/DMS/skills/baogaomiao/scripts/covers`

### 问题3: 类型安全
**问题**: 配置值从 YAML 读取后可能是字符串
**修复**: 添加 `int()` / `float()` 类型转换

## 使用方法

### 方法1: 命令行
```bash
python3 editorial_cover.py \
  --source "来源机构" \
  --page-count 59 \
  --chinese-title "2026标题\n第二行" \
  --english-title "English Title" \
  --year "2026" \
  --summary-text "摘要内容..."
```

### 方法2: Python 调用
```python
from scripts.editorial_cover import EditorialCoverGenerator

generator = EditorialCoverGenerator()
result = generator.generate_cover(
    source="工控网",
    page_count=59,
    chinese_title="2026机器人赛道\n行业竞争分析报告",
    english_title="Robotics Industry Competitive Analysis",
    year="2026",
    summary_text="摘要内容..."
)

print(result['path'])
```

## 配置项说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `max_chinese_chars_per_line` | 8 | 中文标题每行最多字数 |
| `max_english_length` | 25 | 英文标题最多字符数 |
| `max_highlight_title_length` | 60 | 摘要标题最多字数 |
| `max_summary_text_length` | 80 | 摘要内容最多字数 |
| `container_width` | 800 | 封面容器宽度（px） |
| `container_height` | 1066 | 封面容器高度（px，3:4比例） |
| `page_num_font_scale` | 2.0 | 页码字号缩放倍数（基准 3.35rem） |
| `title_font_scale` | 1.0 | 标题字号缩放倍数（基准 3.6rem） |
| `default_output_dir` | covers/ | 默认输出目录 |
