使用全角竖线 `｜` 分隔3行内容：

```bash

```

**关键点**：
- 使用全角竖线 `｜`（不是半角 `|`）
- 格式：`年份｜主题｜报告性质`
- 第1行年份会被替换为固定的2026

### 代码逻辑

```python
# 固定使用2026作为年份（不管原标题是哪一年）
year_prefix = "2026"

# 优先处理｜断句
if '｜' in title_content:
    parts = title_content.split('｜')
    if len(parts) >= 3:
        # 忽略原标题的年份，只提取第2、3部分
        line2 = parts[1].strip()
        line3 = parts[2].strip()
        title_lines_html = f"{year_prefix}<br>{line2}<br>{line3}"
```

## ⚠️ 注意事项

1. **竖线类型**：必须使用全角竖线 `｜`（U+FF5C），不是半角 `|`（U+007C）
2. **固定年份**：不管原标题是哪一年，封面统一显示2026
3. **3行格式**：封面只显示3行中文，英文标题在底部
4. **断句数量**：必须提供3部分（年份、主题、性质），用 `｜` 分隔
