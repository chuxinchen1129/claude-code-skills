# 报告喵系统 - 完整使用指南

## 📋 系统概述

报告喵是一个自动化研报处理系统,包含以下功能:

1. **东方财富研报爬虫** - 自动爬取最新研报
2. **智能封面生成** - 社论风小红书封面
3. **飞书推送** - 自动发送到飞书群
4. **定时任务** - 每天自动运行

---

## 🚀 快速开始

### 方式1: 手动运行(推荐用于测试)

```bash
# 1. 运行东方财富爬虫
cd ~/Desktop/DMS

# 卸载定时任务
./bin/schedule_manager.sh uninstall
```

## 📊 核心功能详解

### 1. 东方财富爬虫

**功能**:
- 自动爬取东方财富最新研报
- 过滤页数<20的报告
- 自动发送清单到飞书

**使用方法**:

```bash
# 爬取最近3天的报告
python3 bin/eastmoney_crawl_and_notify.py --days 3

# 爬取最近1天的报告(测试用)
python3 bin/eastmoney_crawl_and_notify.py --days 1
```

**输出位置**:
- PDF文件: `~/MediaCrawler/MediaCrawler/data/eastmoney/pdf/`
- 元数据: `~/MediaCrawler/MediaCrawler/data/eastmoney/pdf/*.json`
- 日志: `~/Desktop/DMS/temp/logs/eastmoney_crawl.log`

**飞书交互**:
爬取完成后会自动发送清单到飞书,格式:
```
📊 东方财富新下载报告清单(共X份,页数≥20):

1. 国金证券-家电材料基石稳固-32页
2. 开源证券-母婴连锁经营数据报告-45页
...

💬 回复序号选择报告(如:1),或回复「全部」
```

### 2. 报告移动和处理

**步骤1: 选择报告**

在飞书中回复序号:
- `1` - 选择第1份报告
- `1,2,3` - 选择多份报告
- `全部` 或 `all` - 选择全部

**步骤2: 移动到报告喵**

报告会自动移动到:
- 待选报告: `~/Library/Mobile Documents/com~apple~CloudDocs/家人共享/待选报告/`
- 报告喵: `~/Library/Mobile Documents/com~apple~CloudDocs/家人共享/报告喵/`

### 3. 报告喵处理流程

**完整工作流**:

```bash
# 方法1: 通过任务文件(推荐)
# 1. 创建任务文件(由bot_server或手动创建)
# 2. 运行任务处理器
python3 skills/baogaomiao/scripts/baogaomiao_task_processor.py --check

# 方法2: 直接在Claude Code中说"报告喵"
# 我会自动处理PDF,生成笔记和封面
```

**处理步骤**:
1. ✅ 提取PDF文本和截图
2. ✅ 生成小红书笔记(中文标题+英文标题+正文)
3. ✅ 生成社论风封面(HTML + PNG)
4. ✅ 发送到飞书(文本 + 截图 + 封面)

**输出位置**:
- 截图: `~/Desktop/DMS/skills/baogaomiao/screenshots/`
- 封面HTML: `~/Desktop/DMS/skills/baogaomiao/scripts/covers/*.html`
- 封面PNG: `~/Desktop/DMS/skills/baogaomiao/scripts/covers/*.png`

### 4. 封面生成规则

**3行标题格式**:
```
2026
中国移动互联网
流量年度报告
```

**生成规则**:
- 第1行: 固定年份2026
- 第2行: 主题名称
- 第3行: 报告类型

**支持的分隔符**:
- 全角竖线 `｜`
- 半角竖线 `|`
- 换行符 `\n`

**示例**:
```python
# 方式1: 使用分隔符
chinese_title = "2026｜中国移动互联网｜流量年度报告"

# 方式2: 使用换行
chinese_title = """2026
中国移动互联网
流量年度报告"""
```

**封面参数**:
```python
{
    'source': 'CNNIC',           # 研究机构
    'page_count': 74,            # 页数
    'chinese_title': '2026｜中国移动互联网｜流量年度报告',
    'english_title': 'CHINA MOBILE INTERNET TRAFFIC REPORT',
    'year': '2026',              # 固定2026
    'highlight_title': 'AI重塑流量格局',   # 核心观点(最多12字)
    'summary_text': '流量见顶 AI重塑 场景化应用',  # 关键词(最多40字)
    'report_type': '行业研究报告',
    'number': '0312'             # 日期后缀
}
```

## 🔧 高级功能

### 1. PDF重命名

自动将PDF重命名为标准化格式:

```python
from skills.baogaomiao.scripts.file_namer import PDFRenamer

renamer = PDFRenamer()
new_name = renamer.generate_filename("🎯2026 母婴连锁经营数据报告")
# 输出: "母婴连锁经营数据.pdf"

# 执行重命名
result = renamer.rename_pdf(
    old_path="/path/to/original.pdf",
    new_name=new_name
)
```

**命名规则**:
- 移除emoji和年份
- 移除副标题(——之后的内容)
- 截取前12个字符
- 添加机构前缀(如有)

### 2. 任务处理器

**守护进程模式**:
```bash
# 持续检查并处理新任务
python3 skills/baogaomiao/scripts/baogaomiao_task_processor.py \
    --daemon \
    --interval 60 \
    --enable-screenshots
```

**任务文件格式**:
```json
{
    "task_type": "baogaomiao_generate",
    "pdf_path": "/path/to/report.pdf",
    "task_id": "20260312_092926",
    "created_at": "2026-03-12T09:29:26",
    "status": "pending"
}
```

**任务状态**:
- `pending` - 待处理
- `processing` - 处理中
- `completed` - 已完成
- `failed` - 失败

## 🐛 常见问题

### Q1: 定时任务没有运行

**检查步骤**:
```bash
# 1. 查看定时任务状态
cd ~/Desktop/DMS
./bin/schedule_manager.sh status

# 2. 查看日志
tail -f ~/Desktop/DMS/temp/logs/eastmoney_crawler.stdout.log
tail -f ~/Desktop/DMS/temp/logs/eastmoney_crawler.stderr.log

# 3. 手动运行测试
./bin/schedule_manager.sh run
```

**常见原因**:
- LaunchDaemon未加载 → 运行 `./bin/schedule_manager.sh install`
- 日志目录不存在 → 运行 `mkdir -p ~/Desktop/DMS/temp/logs`
- PDF下载目录权限问题 → 检查 `~/MediaCrawler/MediaCrawler/data/eastmoney/pdf/` 权限

### Q2: 封面生成失败

**检查步骤**:
```bash
# 1. 检查配置文件
cat ~/Desktop/DMS/skills/baogaomiao/scripts/editorial_cover_config.yaml

# 2. 检查截图工具
which we2python

# 3. 手动测试封面生成
python3 << EOF
from skills.baogaomiao.scripts.editorial_cover import EditorialCoverGenerator

gen = EditorialCoverGenerator()
result = gen.generate_cover(
    source='CNNIC',
    page_count=74,
    chinese_title='2026｜中国移动互联网｜流量年度报告',
    english_title='CHINA MOBILE INTERNET TRAFFIC REPORT',
    year='2026',
    highlight_title='AI重塑流量格局',
    summary_text='流量见顶 AI重塑'
)

print(f"Success: {result['success']}")
print(f"Path: {result.get('path')}")
EOF
```

**常见原因**:
- 中文标题格式不正确 → 检查是否使用`｜`或`\n`分隔
- 配置文件缺失 → 重新生成配置文件
- 截图工具未安装 → 安装 `we2python`

### Q3: 飞书发送失败

**检查步骤**:
```bash
# 1. 检查飞书配置
cat ~/.feishu_user_config.json

# 2. 检查chat_id
# 确保chat_id不为空

# 3. 测试发送
python3 << EOF
from skills.feishu_universal.scripts.feishu_bot_notifier import FeishuBotNotifier

notifier = FeishuBotNotifier()
result = notifier.send_message("测试消息")
print(f"Success: {result}")
EOF
```

**常见原因**:
- `chat_id`为空 → 从飞书群设置中获取chat_id并更新配置
- token过期 → 重新运行飞书认证流程
- 网络问题 → 检查网络连接

### Q4: PDF重命名失败

**错误信息**:
```
⚠️ 重命名异常: 'PosixPath' object has no attribute 'endswith'
```

**解决方案**:
已在`baogaomiao_task_processor.py`第131行修复,确保调用时只传入`chinese_title`参数:

```python
# 错误写法
new_filename = renamer.generate_filename(chinese_title, Path(pdf_path))

# 正确写法
new_filename = renamer.generate_filename(chinese_title)
```
