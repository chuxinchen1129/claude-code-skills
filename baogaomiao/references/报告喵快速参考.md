## 🎨 封面生成格式

### 3行标题格式
```python
chinese_title = "2026｜中国移动互联网｜流量年度报告"
```

**分隔符支持**: `｜` 或 `|` 或 `\n`

**参数说明**:
- `source`: 研究机构名称
- `page_count`: 报告页数
- `year`: 固定为 "2026"
- `highlight_title`: 核心观点(最多12字)
- `summary_text`: 关键词(最多40字)
- `report_type`: 报告类型
- `number`: 日期后缀(如 "0312")

## 🔍 问题排查

### 定时任务没运行?
```bash
./bin/schedule_manager.sh status
tail -f ~/Desktop/DMS/temp/logs/eastmoney_crawler.stdout.log
```

### 封面生成失败?
```bash
# 检查配置
cat ~/Desktop/DMS/skills/baogaomiao/scripts/editorial_cover_config.yaml

# 检查截图工具
which we2python
```

### 飞书发送失败?
```bash
# 检查配置
cat ~/.feishu_user_config.json

# 确保chat_id不为空
```

### PDF重命名失败?
```bash
# 已在baogaomiao_task_processor.py:131修复
# 确保只传入chinese_title参数
```

## 📊 工作流程图

```
1. 定时任务(8:00)
   ↓
2. 运行爬虫 → 下载PDF到 ~/MediaCrawler/.../pdf/
   ↓
3. 发送清单到飞书
   ↓
4. 用户在飞书中选择序号
   ↓
5. 移动到待选报告/
   ↓
6. 用户再次选择 → 移动到报告喵/
   ↓
7. 创建任务文件
   ↓
8. 任务处理器自动处理:
   - 提取PDF文本和截图
   - 生成小红书笔记
   - 生成社论风封面
   - 发送到飞书
```
