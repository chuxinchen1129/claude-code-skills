# 小红书调研 Skill

## 功能概述

自动采集小红书特定关键词的图文和视频数据，计算互动量（点赞+收藏×2+评论×3+分享×4），按互动量排序输出TOP结果，自动下载图片和视频并按标题命名，生成Excel数据表和MD分析报告。

---

## 使用场景

- **采集小红书数据**：当用户提到"小红书调研"、"采集XX关键词"
- **生成调研报告**：提供数据概览、卖点分析、创作建议
- **定时分析**：配合 daily-review 进行每周/每日自动分析

---

## 快速开始

```bash
# 基本用法
python -m skills.xiaohongshu-research --keywords "茯苓" --count 30

# 清空缓存后采集
python -m skills.xiaohongshu-research --keywords "茯苓" --count 30 --clear-cache

# 自定义日期
python -m skills.xiaohongshu-research --keywords "茯苓" --date 2026-03-01
```

---

## 参数说明

| 参数 | 简写 | 类型 | 默认值 | 说明 |
|------|------|------|---------|------|
| `--keywords` | `-k` | 字符串 | 必需 | 搜索关键词 |
| `--count` | `-c` | 整数 | 30 | 采集数量 |
| `--date` | `-d` | 字符串 | 当天 | 任务日期（YYYY-MM-DD），用于输出目录命名 |
| `--clear-cache` | - | 布尔 | False | 是否清空MediaCrawler缓存 |
| `--help` | `-h` | - | - | 显示帮助信息 |

│         │         │            │    │
│    ┌────▼────┐ │     ┌───────▼─────┐ │
│    │Crawler  │ │     │Processor    │ │
│    │         │ │     │             │ │
│    └────┬────┘ │     └──────┬───────┘ │
│         │        │             │            │

### 详细步骤

1. **解析参数**：读取命令行参数，设置默认值

2. **清空缓存**（可选）：
   - 删除 MediaCrawler 数据目录中的旧JSON文件
   - 删除 images/ 和 videos/ 目录

3. **调用MediaCrawler采集**：
   - 修改 `~/MediaCrawler/config/base_config.py`（KEYWORDS, CRAWLER_MAX_NOTES_COUNT）
   - 执行 `uv run main.py`
   - 等待采集完成
   - 读取 `采集数据/json/search_contents_*.json`

4. **数据处理**：
   - 筛选包含关键词的笔记
   - 计算互动总分（点赞+收藏×2+评论×3+分享×4）
   - 按互动量降序排列
   - 取TOP N 条

5. **媒体处理**：
   - 从 MediaCrawler images/ 复制匹配的图片文件夹
   - 从 MediaCrawler videos/ 复制匹配的视频
   - 重命名格式：`{标题}_{note_id}`（标题在前，note_id在后）
   - 保存到每日工作/data/temp/（临时）

6. **生成Excel**：
   - 使用 openpyxl 创建数据表
   - 包含所有必需列
   - 保存到 `每日工作/{日期}_小红书调研_{关键词}/data/{日期}_小红书调研_{关键词}数据.xlsx`

7. **生成MD报告**：
   - 统计数据概览
   - 提取TOP10高互动内容
   - 分析核心卖点（关键词频率、受欢迎程度）
   - 分析标题特征
   - 提供内容创作建议
   - 保存到 `每日工作/{日期}_小红书调研_{关键词}/reports/{日期}_小红书调研_{关键词}报告.md`

8. **清理临时文件**：
   - 删除 champ_*.pyc
   - 删除 __pycache__ 目录
   - 清理 temp/ 目录

## 常见问题

### Q: 采集失败怎么办？

检查 MediaCrawler 是否可正常访问，网络是否连接正常，重新运行脚本。

### Q: 如何只下载图片不下载视频？

MediaCrawler 会同时下载图片和视频，脚本会自动处理。

### Q: 数据量不足怎么办？

可能是关键词搜索结果不足，尝试更换相关关键词或增加搜索范围。

### Q: 文件名包含特殊字符怎么办？

脚本会自动清理文件名中的非法字符，只保留安全字符。
