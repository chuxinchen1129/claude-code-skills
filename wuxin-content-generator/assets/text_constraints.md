# 文本限制规范

> **目的**: 确保品牌一致性和内容质量
> **核心原则**: 避免AI味、避免过度营销、保持品牌定位

---

## 负向约束（禁止）

### 1. 禁止销售话术

**❌ 禁用词汇**：
- "限时优惠"
- "马上抢购"
- "错过再等一年"
- "买一送一"
- "史无前例"
- "超级优惠"
- "最后机会"
- "不容错过"

**✅ 替代表达**：
- "新年首发价，直降200元"
- "仅需1099元"
- "30天无理由退换"

---

### 2. 禁止重复词组

**❌ AI句式**：
- "不是…而是…"
- "不仅…而且…"
- "一方面…另一方面…"
- "无论是…还是…"

**✅ 替代表达**：
- 直接陈述，避免套句
- "悟昕采用脑电波监测技术"（而不是"不是心率监测，而是脑电波监测"）
- "悟昕具有监测和干预双重功能"（而不是"不仅能监测，而且能干预"）

---

### 3. 禁止堆砌形容词

**❌ 错误示例**：
- "显著提升、明显改善、有效解决"
- "强大的、专业的、先进的、创新的"
- "非常好、特别棒、超级厉害"

**✅ 替代表达**：
- "准确率达医疗级设备的80%以上"
- "缩短30%入睡时间"
- "睡眠质量改善率>75%"

---

### 4. 禁止论文式连接

**❌ 禁用词汇**：
- "在当今时代"
- "综上所述"
- "值得注意的是"
- "随着科技的发展"
- "在这个快节奏的社会"
- "首先、其次、再次、最后"

**✅ 替代表达**：
- 直接切入主题
- 使用情感化开头
- 场景化引入

---

### 5. 禁止机械列表

**❌ 错误示例**：
```
悟昕有以下优点：
1. 监测准确
2. 干预精准
3. 效果显著
4. 使用方便
```

**✅ 替代表达**：
```
悟昕睡眠仪，三大核心优势：
- 它真的准：医疗级脑电波监测，直达睡眠核心
- 它真的懂：AI个性化干预，为你量身定制
- 它真的舒服：眼罩形态，轻量化设计
```

## 品牌定位约束

### 1. 智能设备品牌

**✅ 可以说**：
- "医疗级脑电波监测"
- "非药物物理助眠"
- "智能睡眠管理设备"
- "AI睡眠解决方案"

**❌ 禁止说**：
- 镁片
- 褪黑素
- 维生素
- 营养品
- 保健品
- 膳食补充剂
- 营养补充剂

### 2. 禁止提及竞品

**❌ 禁止**：
- "比苹果手表更准确"
- "比华为手环更专业"
- "比小米手环更好"

**✅ 正确对比**：
- "不像手环只是测心率"
- "不同于市面上的心率估算"
- "告别传统穿戴设备的猜测"

---

## 平台合规约束

### 1. 禁用绝对化词汇

**❌ 禁用**：
- "治疗"
- "根治"
- "第一"
- "最好"
- "顶级"
- "完美"
- "保证"
- "100%有效"
- "完全治愈"
- "无副作用"

**✅ 替代表达**：
- "改善"睡眠
- "辅助"入眠
- "缓解"焦虑
- "监测"睡眠数据
- "安全性高"
- "有效改善"
- "帮助"入睡

### 2. 数据必须真实

**✅ 有来源的数据**：
- 80%+：医疗级准确率（内部实验室验证）
- 30%：入睡时间缩短（实验室验证）
- 75%：睡眠质量改善率（临床数据）
- 48%：成年人失眠率（《2025中国睡眠健康调查报告》）
- 3亿：睡眠障碍人数（《2025中国睡眠健康调查报告》）

**❌ 禁止编造**：
- 所有数据必须有来源支撑
- 禁止夸大效果
- 禁止模糊表达

## 内容结构约束

### 小红书：20-60-20法则

**Intro（20%）**：30-50字
- 痛点开头
- 场景代入
- 情感共鸣

**3板块（60%）**：各150字
- 板块1：科学原理
- 板块2：悟昕方案
- 板块3：使用感受

**CTA（20%）**：50-80字
- 行动引导
- 情感升华
- 互动引导

### 公众号：娓娓道来

**结构要求**：
- 痛点开头：情感共鸣
- 原因分析：科学原理
- 产品介绍：自然过渡
- 核心卖点：完整呈现
- 情感升华：价值主张

**字数要求**：2000-3000字

### 短视频：四段式

**0-3s**：黄金钩子（10-15字）
**3-10s**：痛点深挖（25-35字）
**10-20s**：原理展现（35-50字）
**20-30s**：爽感结尾（25-35字）

## 质量检查清单

### AI味检测
- [ ] 是否包含套话（"在当今时代"、"综上所述"）
- [ ] 是否有AI句式（"不是…而是…"）
- [ ] 是否有书面词汇（"显著提升"、"明显改善"）

### 品牌定位检测
- [ ] 品牌调性：温暖治愈、科学循证、优雅克制
- [ ] 品牌定位：智能设备，不提及营养补充剂
- [ ] 核心卖点：7大卖点+3大支柱完整

### 合规检测
- [ ] 禁用词：治疗、根治、第一、最好、保证
- [ ] 推荐词：改善、辅助、缓解、监测、物理助眠
- [ ] 数据真实：所有数据需有来源支撑
