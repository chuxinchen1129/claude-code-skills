# 一天光线时间轴 - 光照暴露指南

> 基于 Huberman Lab 科学研究，精确规划一天中的光照暴露，优化昼夜节律和睡眠质量。

## 🌅 早晨光照（6:00-10:00）

### 醒来立即光照 ⭐ 最重要

**最佳时间：** 起床后 30-60 分钟内
**持续时间：** 10-30 分钟
**强度要求：** 至少 2,500-10,000 lux

**获取方式：**
| 天气 | 获取方式 | 时长 |
|------|----------|------|
| 晴天 | 直接户外活动 | 10 分钟 |
| 多云 | 户外活动（无需直射） | 20 分钟 |
| 雨天 | 户外 + 靠窗位置 | 30 分钟 |
| 无法户外 | 10,000 lux 光疗灯 | 20-30 分钟 |

**关键原则：**
- ✅ **越大越好：** 眼睛直接面向光源，不戴太阳镜
- ✅ **越早越好：** 延迟光照会推迟昼夜节律
- ✅ **每天坚持：** 即使阴天也有足够光线
- ❌ **避免：** 通过窗户或汽车玻璃（阻挡 50% 以上光线）

### 皮质醇自然上升

**早晨光照的作用：**
```
光线 → 视网膜 → 下丘脑 → 皮质醇分泌 → 清醒 & 精力
```

**避免早晨咖啡因的原因：**
- 让皮质醇自然上升
- 咖啡因过早上场会干扰这一过程
- 醒后 60-90 分钟再喝咖啡最佳

## ☀️ 上午光照（10:00-14:00）

### 工作学习时段

**目标：** 维持高水平警觉性

**策略：**
- 将工作台放置在靠近窗户的位置
- 每小时休息时接触自然光 5 分钟
- 午餐前后户外散步 10-15 分钟

**办公室优化：**
```
无窗办公室 → 每小时户外 5-10 分钟
靠窗位置 → 自然光充足，充分利用
人工照明 → 确保足够亮度（500+ lux）
```

### 温度最低点后光照

**温度最低点：** 通常在习惯性起床前 2 小时

**光照策略：**
- ✅ 温度最低点**之后**的光照 → 推进时钟（更容易早起）
- ❌ 温度最低点**之前**的光照 → 推迟时钟（更容易晚睡）

**实际应用：**
- 想养成早起习惯 → 醒来后立即强光
- 想推迟就寝时间 → 傍晚强光照射

## 🌆 下午光照（14:00-17:00）

### 避免午后崩溃

**下午困倦的生理原因：**
```
午餐后血糖波动 → 皮质醇自然下降 → 困意来袭
```

**光照对策：**
- [ ] 下午 2-3 点户外活动 10-15 分钟
- [ ] 如果无法户外，使用光疗灯
- [ ] 避免长时间在昏暗环境中

**咖啡因配合：**
- 下午 2 点前最后一杯咖啡因
- 与光照配合，效果最佳

## 🌇 傍晚光照（17:00-20:00）

### 太阳能板效应

**黄金时间：** 日落前 1 小时

**光照目的：**
- 强化昼夜节律
- 为睡眠做准备
- 防止季节性情感障碍（SAD）

**获取建议：**
| 天气 | 时长 | 方式 |
|------|------|------|
| 晴天 | 20-30 分钟 | 户外散步或活动 |
| 多云 | 30-45 分钟 | 延长户外时间 |
| 冬季/高纬度 | 45-60 分钟 | 最大程度获取 |

## 🌆 睡前光照管理（20:00-22:00）

### 调暗阶段 ⚠️ 关键时期

**睡前 1-2 小时：**
- [ ] 将室内灯光调暗 50% 或更多
- [ ] 使用暖色调灯光（2700K 或更低）
- [ ] 关闭不必要的照明
- [ ] 避免使用电子设备或使用蓝光过滤器

**蓝光危害：**
```
蓝光 → 抑制褪黑素 → 延迟入睡 → 睡眠质量下降
```

**夜间屏幕策略：**
| 时间 | 策略 |
|------|------|
| 睡前 2 小时 | 理想：完全避免 |
| 睡前 1 小时 | 可接受：启用夜视模式 |
| 睡前 30 分钟 | 避免：任何屏幕使用 |

## 🌙 睡眠期间（22:00-6:00）

### 完全黑暗环境

**为什么需要完全黑暗？**
- 任何光线都会抑制褪黑素
- 即使微弱光线也会影响睡眠质量
- 完全黑暗促进深度睡眠

**实现方法：**
1. **遮光窗帘**（阻挡 95%+ 光线）
2. **睡眠眼罩**（舒适且完全遮光）
3. **移除所有发光设备**（LED 灯、充电器）
4. **门缝挡光条**（阻挡走廊光线）

## 🌅 早晨循环重复

### 新的一天开始

**起床后立即：**
1. **拉开窗帘**，让阳光进入
2. **走出户外**，直接接触自然光
3. **如需人工光源**，使用 10,000 lux 光疗灯

## 📊 季节调整

### 夏季（长日照）

```
日出早，日落晚 → 光照充足，容易获取
重点：避免傍晚光线，调暗环境
```

**调整策略：**
- [ ] 使用遮光窗帘应对晚日落
- [ ] 睡前 2 小时调暗灯光
- [ ] 早晨光照非常充足，充分利用

### 冬季（短日照）

```
日出晚，日落早 → 光照不足，节律紊乱风险
重点：主动获取光照，使用人工光源
```

**调整策略：**
- [ ] 早晨立即使用光疗灯 20-30 分钟
- [ ] 工作时间尽可能靠近窗户
- [ ] 午休时户外活动
- [ ] 考虑维生素 D 补充

### 阴雨天气

```
云层阻挡 50%+ UVB，但可见光仍充足
重点：延长户外时间，坚持获取光照
```

**应对方法：**
- [ ] 不要因为有云就放弃户外活动
- [ ] 延长户外时间 2-3 倍
- [ ] 光线仍然充足，只是感觉暗

## 🎯 特殊人群调整

|------|----------|
| 通勤车内 | 走路/骑行部分路程 |
| 办公室无窗 | 每小时户外 5 分钟 |
| 久坐不动 | 站立工作，户外午餐 |

### 倒班族

| 时间段 | 光照策略 |
|--------|----------|
| 班前（傍晚） | 强光照射，保持清醒 |
| 班中（深夜） | 环境光照保持警觉 |
| 班后（清晨） | 戴墨镜避免强光，准备睡眠 |

### 老年人

| 挑战 | 解决方案 |
|------|----------|
| 晶状体黄变 | 需要更亮、更久的光照 |
| 活动受限 | 尽可能坐近窗户 |
| 户外限制 | 光疗灯是必需品 |

## 🚀 光照优化工具包

### 推荐设备

**光疗灯：**
- 功率：10,000 lux
- 用途：阴天、冬季、早晨快速唤醒
- 使用：20-30 分钟，距离 30-60 厘米

**蓝光过滤器：**
- 应用软件：f.lux、Night Shift
- 眼镜：黄色/琥珀色镜片
- 屏幕：夜间模式或护眼模式

**光照监测器：**
- 应用：Lux Light Meter
- 目标：确认户外光照强度
- 阈值：早晨 > 2,500 lux
