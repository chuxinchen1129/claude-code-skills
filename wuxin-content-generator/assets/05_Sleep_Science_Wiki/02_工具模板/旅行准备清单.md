# 旅行准备清单

> 跨时区旅行前、中、后的完整准备和应对方案。

## 📋 旅行前准备（出发前 1-2 周）

### 确定旅行类型

| 旅行方向 | 时差调整策略 | 关键措施 |
|----------|--------------|----------|
| **向东**（需要早起） | 提前就寝，提前起床 | 早晨强光，傍晚避光 |
| **向西**（需要晚睡） | 推后就寝，推后起床 | 傍晚强光，早晨避光 |
| **≤ 2 时区** | 最小调整 | 保持原有作息即可 |
| **3-5 时区** | 中度调整 | 逐步调整，2-3 天适应 |
| **≥ 6 时区** | 重大调整 | 提前 1 周开始准备 |

### 预调整方案

**向东旅行（如中国 → 欧洲）：**

**出发前 3-5 天：**
- [ ] 每天将就寝时间提前 30-60 分钟
- [ ] 每天将起床时间提前 30-60 分钟
- [ ] 早晨立即接触明亮自然光
- [ ] 傍晚调暗灯光，避免强光

**向西旅行（如中国 → 美国）：**

**出发前 3-5 天：**
- [ ] 每天将就寝时间推后 30-60 分钟
- [ ] 每天将起床时间推后 30-60 分钟
- [ ] 傍晚接触明亮自然光
- [ ] 早晨避免强光（戴墨镜）

### 准备物品清单

**机上/火车睡眠用品：**
- [ ] 颈枕（U 型枕）
- [ ] 眼罩（遮光必备）
- [ ] 耳塞或降噪耳机
- [ ] 薄外套或毯子（机舱温度低）
- [ ] 舒适的衣物（宽松、透气）
- [ ] 保湿用品（小风扇、保湿喷雾）

**补充剂包：**
- [ ] 褪黑素 3-5mg
- [ ] 镁 200-400mg
- [ ] L-茶氨酸 100-200mg
- [ ] 电解质粉包
- [ ] 维生素 C 和 D

## ✈️ 旅行途中策略

### 长途飞行（6+ 小时）

**机上睡眠策略：**

1. **向东飞行（如中国 → 欧洲）**
   - ✅ 尽量在机上睡觉
   - ✅ 使用眼罩和耳塞
   - ✅ 避免机上屏幕蓝光
   - ❌ 除非是当地夜晚，否则避免睡觉

2. **向西飞行（如中国 → 美国）**
   - ❌ 避免在机上睡觉
   - ✅ 保持清醒，看电影/阅读
   - ✅ 避免机上晚餐，保持空腹
   - ❌ 到达前不要睡觉

### 补水与饮食

**飞行中的黄金法则：**

| 行为 | 频率 | 原因 |
|------|------|------|
| 喝水 | 每 1-2 小时 | 对抗机舱干燥 |
| 避免酒精 | 完全避免 | 加重脱水，破坏睡眠 |
| 限制咖啡因 | 飞行前半段 | 影响在飞机上睡觉 |
| 轻食 | 少量多餐 | 避免消化不良 |
| 活动 | 每 1-2 小时走动 | 促进血液循环 |

## 🌅 到达后适应策略

### 向东到达后（如到达欧洲）

**目标：**比习惯时间早起

**第 1 天（到达日）：**
- [ ] **拒绝睡觉！** 即使是傍晚到达
- [ ] 接触明亮自然光（最重要）
- [ ] 参加户外活动
- [ ] 晚上 7-8 点后调暗灯光
- [ ] 当地晚上 9-10 点就寝

**第 2 天：**
- [ ] 当地时间 6-7 点起床，接触强光
- [ ] 咖啡因仅限于上午
- [ ] 白天保持活动，避免午睡
- [ ] 晚上保持规律作息

**向东旅行适应技巧：**
- 早餐在户外明亮阳光下
- 避免傍晚强光照射
- 第一天晚上可能较早困倦，顺应它

### 向西到达后（如到达美国）

**目标：**比习惯时间晚睡

**第 1 天（到达日）：**
- [ ] 按当地时间作息（可能还早）
- [ ] 白天保持适度活动
- [ ] 午睡限制在 20-30 分钟
- [ ] 下午避免强光（戴墨镜）
- [ ] 傍晚接触明亮自然光
- [ ] 尽可能晚睡

**第 2 天：**
- [ ] 自然醒来（可能很早）
- [ ] 早晨避免强光，戴墨镜
- [ ] 午睡可延长至 60 分钟
- [ ] 下午接触明亮光照
- [ ] 推迟就寝时间

**向西旅行适应技巧：**
- 上午避免强光（戴墨镜）
- 下午寻求强光照射
- 第一天可能很早醒来，保持低刺激活动

## 💊 补充剂支持

### 飞行中/到达后使用

**褪黑素（向东飞行）：**
- **剂量：** 3-5mg
- **时机：** 当地就寝时间前 60 分钟
- **目的：** 帮助调整昼夜节律

**褪黑素（向西飞行）：**
- **剂量：** 0.5-3mg
- **时机：** 当地睡眠时间（如果太早）前 60 分钟
- **目的：** 延长睡眠，直到适应

**支持性补充剂：**
- **镁：** 200-400mg，睡前，放松肌肉
- **L-茶氨酸：** 100-200mg，睡前，减轻焦虑
- **电解质：** 飞行中和到达后，对抗脱水

## 🚨 旅行者睡眠问题应对

### 问题 1：到达后夜间觉醒

**原因：** 昼夜节律尚未完全调整

**应对：**
- [ ] 不要看时间，减少焦虑
- [ ] 使用 20 分钟法则
- [ ] 进行 NSDR 或冥想练习
- [ ] 接受短暂的睡眠中断

### 问题 2：到达后白天过度困倦

**原因：** 睡眠不足 + 节律失调

**应对：**
- [ ] 限制午睡至 20-30 分钟
- [ ] 自然光照是最有效的提神剂
- [ ] 咖啡因仅限于上午使用
- [ ] 轻度运动（15-30 分钟散步）

### 问题 3：到达后消化问题

**原因：** 时差影响消化节律

**应对：**
- [ ] 到达后 24 小时清淡饮食
- [ ] 避免大量油腻食物
- [ ] 补充益生菌和电解质
- [ ] 适当禁食 12-16 小时帮助重置

## ✅ 旅行前最后检查

**出发前 24 小时：**
- [ ] 充足睡眠（不要熬夜准备）
- [ ] 避免酒精和过量咖啡因
- [ ] 完成所有重要工作
- [ ] 准备机上睡眠用品
- [ ] 装载 NSDR 音频或冥想应用
- [ ] 下载离线娱乐内容
- [ ] 准备补充剂（分装到小袋子）

**出发时：**
- [ ] 按照新时区的作息开始调整（如果可能）
- [ ] 保持正向心态："几天后就会适应"
- [ ] 接受可能的睡眠中断
