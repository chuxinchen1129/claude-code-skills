## 📝 内容创作流程

### 标准流程
1. **查看marketing_calendar** → 确定当前营销节点
2. **检索话题库** → 选择合适话题
3. **参考创作指南** → 确定内容结构和角度
4. **引用核心知识** → 获取科学背书和数据
5. **整合品牌资产** → 植入产品卖点
6. **优化SEO** → 使用热门关键词

### 20-60-20 内容结构公式
- **20% 开头**：反直觉观点 + 情感共鸣
- **60% 主体**：科学解释 + 数据支撑 + 实用建议
- **20% 结尾**：行动号召 + 引导互动
