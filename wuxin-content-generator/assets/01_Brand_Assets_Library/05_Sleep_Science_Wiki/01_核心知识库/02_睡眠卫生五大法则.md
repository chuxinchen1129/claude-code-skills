# 睡眠卫生五大法则

> Huberman Lab 总结的五个睡眠卫生核心原则，遵循这些法则可以显著改善睡眠质量。

## 🌙 法则一：Regularity（规律性）

**核心原则：每天（包括周末）在同一时间上床和起床。**

### 为什么规律性最重要？

```
规律性 = 昼夜节律稳定性 = 激素分泌规律 = 优质睡眠
```

**科学依据：**
- 24 小时节律时钟需要稳定信号来同步
- 不规律的睡眠时间导致"社交时差"
- 周末补觉会破坏节律，产生类似倒时差的效果

### 执行方案

**第一步：固定起床时间 ⏰**
- 选择一个可以每天坚持的起床时间
- 即使前一天睡得晚，也要在同一时间起床
- 起床后立即接触明亮光线

**第二步：就寝时间自然调整**
- 不要强迫自己在特定时间入睡
- 让困意引导就寝时间
- 初期可能存在睡眠债务，会自然调整

**可接受的变动范围：**
- 理想：±15 分钟
- 可接受：±30 分钟
- 避免：±1 小时以上

### 周末陷阱

| 行为 | 后果 |
|------|------|
| 周末晚起 2 小时 | 社交时差，周一早晨困难 |
| 周末晚睡晚起 | 节律完全破坏 |
| 午睡 > 90 分钟 | 晚上睡意延迟 |

## 🌗 法则二：Darkness（黑暗）

**核心原则：睡前 1 小时将灯光调暗 50% 或更多，创造黑暗睡眠环境。**

### 黑暗的生物学机制

**褪黑素分泌曲线：**
```
光亮 → 视网膜内在光敏神经节细胞 → 抑制褪黑素
黑暗 → 褪黑素开始分泌（约需 2 小时达到峰值）
```

### 执行方案

**阶段 1：睡前 1-2 小时**
- 将室内灯光调暗 50%
- 使用暖色调灯光（2700K 或更低）
- 关闭不必要的灯具

**阶段 2：睡前 30 分钟**
- 只使用床头灯或夜灯
- 避免使用任何电子设备
- 保持光源低且远离视线

**阶段 3：睡眠期间**
- 使用遮光窗帘或睡眠眼罩
- 移除或遮盖所有发光设备（LED 灯、充电器指示灯）
- 保持卧室完全黑暗

### 蓝光特别提醒

⚠️ **蓝光是最强褪黑素抑制剂**
- 手机、平板、电脑屏幕富含蓝光
- 睡前 1 小时应避免使用
- 如必须使用，启用蓝光过滤器或夜视模式

## 🌡️ 法则三：Temperature（温度）

**核心原则：身体核心温度下降 1-3°C 才能启动睡眠，室温约 19-20°C 最理想。**

### 体温与睡眠的关系

**睡眠启动机制：**
```
夜间体温下降 → 信号传递给大脑 → 启动睡眠程序
早晨体温上升 → 信号传递给大脑 → 醒来
```

### 执行方案

**环境温度设置：**
- **推荐室温**：19-20°C (67-68°F)
- **过高**：>22°C 会干扰深度睡眠
- **过低**：<16°C 可能导致频繁觉醒

**身体温度调节技巧：**

1. **睡前降温**（启动睡眠）：
   - 睡前 60-90 分钟洗温水浴或淋浴
   - 出来后体温会自然下降，产生睡眠信号
   - 或将卧室温度调低

2. **睡眠中温度管理**：
   - 使用透气的寝具（棉、丝绸）
   - 裸睡或轻薄睡衣
   - 足部保持温暖（穿袜子或使用热水袋）

3. **早晨升温**（促进醒来）：
   - 醒来后立即接触明亮光线
   - 提高室温或进行轻度运动
   - 温暖的淋浴或早餐

### 特殊情况

**热浪天气：**
- 使用风扇或空调
- 冷水浸泡脚部 10 分钟
- 使用凉感枕垫

**寒冬天气：**
- 保暖寝具但保持卧室通风
- 热水袋或电热毯（睡前关闭）
- 保暖睡衣

## 🚶 法则四：Walk It Out（20分钟法则）

**核心原则：醒着时在床上停留不超过 20-25 分钟，起来到另一个房间放松。**

### 为什么要"走出"卧室？

**学习理论原理：**
```
床 = 睡眠的信号
床上清醒时间 = 床与清醒的关联 = 睡眠困难
```

### 执行方案

**何时应用 20 分钟法则？**
- ✅ 入睡困难，躺了 20 分钟还清醒
- ✅ 夜间醒来，无法在 20 分钟内再次入睡
- ✅ 过早醒来，无法再次入睡

**做什么？**
1. 起来，离开卧室
2. 进行低刺激活动：
   - 阅读纸质书（非电子书）
   - 听轻柔音乐或播客
   - 冥想或呼吸练习
   - 温和的伸展
3. **避免**：
   - 明亮光线
   - 电子设备
   - 刺激性内容
   - 吃东西（除了水）

4. 直到有困意再回床上

### 重复循环

如果回到床上 20 分钟仍无法入睡：
```
重复循环：床上 20 分钟 → 起来放松 → 有困意 → 回床上
```

**注意：** 这个过程可能需要重复多次，但值得坚持。

## ☕ 法则五：Substance Management（物质管理）

**核心原则：注意咖啡因和酒精的时机和剂量。**

### 咖啡因管理

**咖啡因代谢时间线：**
```
摄入后 30-60 分钟：达到峰值
5-6 小时（半衰期）：50% 仍留在体内
10-12 小时（四分之一衰期）：25% 仍留在体内
```

**黄金法则：**
- ⏰ **截止时间**：下午 2-3 点前摄入最后一杯咖啡
- 📊 **计算示例**：
  - 早上 8 点喝咖啡 → 晚上 8 点还有 25% 在体内
  - 下午 2 点喝咖啡 → 凌晨 2 点还有 25% 影响睡眠

**个体差异：**
- 快速代谢者：半衰期 2-4 小时
- 慢速代谢者：半衰期可达 10 小时
- 根据自身敏感度调整

### 酒精管理

**酒精 ≠ 助眠剂**

| 误区 | 真相 |
|------|------|
| 酒精帮助入睡 | 酒精是镇静剂，不是睡眠促进剂 |
| 喝一点睡得香 | 酒精碎片化睡眠，导致频繁觉醒 |
| 红酒有助于睡眠 | 酒精强力阻断 REM 睡眠 |

**酒精影响：**
1. **睡眠碎片化**：频繁的不记得的微觉醒
2. **REM 阻断**：影响情绪处理和记忆巩固
3. **呼吸干扰**：加重打鼾和睡眠呼吸暂停
4. **利尿作用**：导致夜间频繁如厕

**安全指南：**
- 🍷 睡前 3-4 小时内避免饮酒
- 📊 如需饮酒，限制在 1-2 标准饮品
- 🌡️ 酒后大量饮水，稀释代谢产物

### 其他物质

**尼古丁：**
- 刺激物，类似咖啡因
- 睡前 2 小时避免吸烟

**大餐：**
- 睡前 2-3 小时避免大量进食
- 消化过程干扰深度睡眠

**水分：**
- 睡前 1 小时减少饮水（避免夜间如厕）
- 白天充分补水

## ✅ 五大法则执行清单

### 每日清单

**☀️ 早晨：**
- [ ] 在固定时间起床（±30 分钟）
- [ ] 立即接触 10-30 分钟明亮自然光
- [ ] 避免在醒后 60-90 分钟内摄入咖啡因

**🌆 日间：**
- [ ] 下午 2-3 点前停止咖啡因
- [ ] 白天充分暴露于自然光
- [ ] 限制午睡在 90 分钟以内（避免下午 3 点后午睡）

**🌆 傍晚：**
- [ ] 晚餐在睡前 3 小时完成
- [ ] 避免酒精和咖啡因
- [ ] 轻度运动或散步

**🌙 睡前 1-2 小时：**
- [ ] 将室内灯光调暗 50%
- [ ] 关闭电子设备或使用蓝光过滤器
- [ ] 开始放松程序（阅读、冥想、温水浴）

**🛏️ 卧室环境：**
- [ ] 室温 19-20°C
- [ ] 完全黑暗（遮光窗帘或眼罩）
- [ ] 安静环境（耳塞或白噪音）

**😴 睡眠中：**
- [ ] 如果 20 分钟无法入睡，起床放松
- [ ] 遵循 20 分钟法则
