# 竞品监控 Skill 使用指南

## 概述

小红书竞品评论监控预警系统，定时扫描"悟昕睡眠仪"相关笔记，检测竞争品牌（强脑、左点、温致）的评论，发现后通过飞书机器人发送预警。

## 功能特性

- ✅ 使用 MediaCrawler 采集真实小红书数据
- ✅ 自动检测评论中的竞品关键词
- ✅ 整合的飞书预警消息（所有竞品评论汇总）
- ✅ 支持定时监控和单次扫描
- ✅ 完整的日志记录

## 配置说明

配置文件: `~/.claude/skills/competitor-alert/config/brands.json`

```json
{
  "keywords": ["悟昕睡眠仪", "悟昕 睡眠仪", "悟昕助眠"],
  "competitors": ["强脑", "左点", "温致"],
  "interval_hours": 2,
  "max_notes": 10,
  "screenshot_enabled": true,
  "comment_min_length": 10,
  "alert_on_brand_mention": true
}
```

### 配置项说明

| 配置项 | 说明 |
|--------|------|
| `keywords` | 要监控的关键词列表 |
| `competitors` | 竞品品牌列表 |
| `interval_hours` | 定时监控间隔（小时） |
| `max_notes` | 每个关键词最多采集笔记数 |
| `screenshot_enabled` | 是否启用截图（预留功能） |
| `comment_min_length` | 评论最小长度 |
| `alert_on_brand_mention` | 是否在品牌提及时预警 |

## 数据采集

### MediaCrawler 集成

本技能使用 MediaCrawler 进行数据采集，需要：

1. **安装 MediaCrawler**
   ```bash
   cd ~
   git clone https://github.com/NanmiCoder/MediaCrawler.git
   cd MediaCrawler
   python3.11 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```

2. **首次使用需要扫码登录**
   - 运行采集时会自动打开浏览器
   - 扫描二维码登录小红书
   - 登录后会保持会话，无需每次扫码

### 数据文件位置

```
~/MediaCrawler/data/xhs/json/
├── search_contents_YYYY-MM-DD.json  # 笔记数据
└── search_comments_YYYY-MM-DD.json  # 评论数据
```

### 配置飞书

本技能使用全局飞书配置: `~/.feishu_user_config.json`

如果没有配置，会尝试使用以下环境变量：
- `FEISHU_APP_ID`
- `FEISHU_APP_SECRET`
- `FEISHU_BOT_WEBHOOK`

## 常见问题

### Q1: MediaCrawler 需要扫码登录怎么办？

A: 这是正常的。首次使用需要扫码登录，登录后会保持会话一段时间。如果登录过期，重新扫码即可。

### Q2: 如何避免频繁扫码？

A: 可以使用 Cookie 登录方式：
1. 登录小红书网页版
2. 获取 Cookie
3. 在 MediaCrawler 配置中设置 Cookie

### Q3: 没有检测到竞品评论？

A: 可能的原因：
- 采集的数据中没有竞品关键词
- 检查配置文件中的 `competitors` 列表是否正确
- 查看日志确认评论是否正确加载

### Q4: 飞书通知没有收到？

A: 检查：
1. `~/.feishu_user_config.json` 是否存在且正确
2. 环境变量是否设置
3. 查看日志中的错误信息
