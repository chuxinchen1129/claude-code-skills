# 飞书交互机器人 v1.2

双向交互的飞书机器人，支持接收用户消息并执行相应操作。

## 🎯 核心功能

### 1. 链接采集 + 智能路由系统 ⭐ ENHANCED

**触发格式**：
- 直接发送链接（支持短链接）
- `采集：关键词 [数量]`

**智能路由系统**（content_router.py）：
- ✅ 自动检测内容类型（小红书视频/图文、微信文章）
- ✅ 短链接自动解析（xhslink.com）
- ✅ 自动提取笔记ID（24位十六进制）
- ✅ 智能路由到对应处理器

**材料组织系统**（material_organizer.py）：
- ✅ 自动保存到"对标参考/"文件夹
- ✅ 按类型分类：小红书视频、小红书图文、微信文章
- ✅ 自动生成可搜索的索引（index.md）
- ✅ 每个笔记独立文件夹（格式：日期_序号_标题）
- ✅ 完整元数据（metadata.json）

### 2. 触发采集

用户发送关键词，触发MediaCrawler采集数据。

**触发格式**：
- `采集：关键词 [数量]`

**参数**：
- 关键词（必需）
- 数量（可选，默认20）

### 3. 数据上传

用户发送指令，将采集的数据上传到飞书表格。

**触发格式**：
- `上传到飞书`
- `上传：项目名称`

### 4. 状态查询

用户查询系统状态和采集结果。

**触发格式**：
- `查看状态`
- `最近采集`
- `统计`

## 🚀 快速开始

### 1. 配置公网服务器

由于飞书需要公网可访问的地址来推送事件，你需要：

**选项A：使用 ngrok（推荐用于测试）**
```bash
# 安装 ngrok
brew install ngrok

# 启动隧道
ngrok http 5000
```

**选项B：使用云服务器**
- 阿里云/腾讯云 ECS
- 配置防火墙开放端口
- 使用域名 + SSL证书

### 2. 配置飞书机器人

1. **获取 App ID 和 App Secret**
   - 登录 [飞书开放平台](https://open.feishu.cn/)
   - 创建企业自建应用
   - 获取凭据

2. **配置事件订阅**
   - 启用 `im.message.receive_v1` 事件
   - 设置 Request URL（你的公网地址 + /webhook）

3. **添加机器人权限**
   - 添加 `im:message` 权限
   - 发送消息权限
   - 添加获取用户信息权限

### 3. 启动机器人服务

```bash
cd ~/.claude/skills/feishu-bot/scripts
python bot_server.py
```
